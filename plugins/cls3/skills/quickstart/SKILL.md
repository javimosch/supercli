# Cls3 Plugin

CLI tool to clear and delete S3 buckets and all objects.

## Commands

### Operations
- `cls3 bucket clear` — clear bucket via cls3
- `cls3 bucket delete` — delete bucket via cls3

## Usage Examples
- "cls3 --help"
- "cls3 <args>"

## Installation

```bash
go install github.com/go-to-k/cls3@latest
```

## Examples

```bash
cls3 --version
cls3 --help
```

## Key Features
- aws\n- s3\n- cleanup
