# Corgi Plugin

Command line tool for project scaffolding and code generation.

## Commands

### Operations
- `corgi project scaffold` — scaffold project via corgi
- `corgi project init` — init project via corgi

## Usage Examples
- "corgi --help"
- "corgi <args>"

## Installation

```bash
go install github.com/junyu-w/corgi@latest
```

## Examples

```bash
corgi --version
corgi --help
```

## Key Features
- scaffold\n- code-generation\n- cli
