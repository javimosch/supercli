# Syncthing Plugin

Continuous file synchronization tool with CLI.

## Commands

### Operations
- `syncthing device list` — list device via syncthing
- `syncthing config dump` — dump config via syncthing

## Usage Examples
- "syncthing --help"
- "syncthing <args>"

## Installation

```bash
brew install syncthing
```

## Examples

```bash
syncthing --version
syncthing --help
```

## Key Features
- file-sync\n- backup\n- p2p
