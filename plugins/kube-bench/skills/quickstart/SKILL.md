# Kube-bench Plugin

CIS Kubernetes Benchmark security scanner.

## Commands

### Operations
- `kube-bench cluster run` — run cluster via kube-bench
- `kube-bench cluster apply` — apply cluster via kube-bench

## Usage Examples
- "kube-bench --help"
- "kube-bench <args>"

## Installation

```bash
brew install kube-bench
```

## Examples

```bash
kube-bench --version
kube-bench --help
```

## Key Features
- kubernetes\n- security\n- cis-benchmark
