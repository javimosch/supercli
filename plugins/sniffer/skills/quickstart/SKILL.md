# Sniffer Plugin

A modern alternative network traffic sniffer.

## Commands

### Operations
- `sniffer interface capture` — capture interface via sniffer

## Usage Examples
- "sniffer --help"
- "sniffer <args>"

## Installation

```bash
brew install sniffer
```

## Examples

```bash
sniffer --version
sniffer --help
```
