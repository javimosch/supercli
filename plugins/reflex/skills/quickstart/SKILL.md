# Reflex Plugin

Run a command when files change.

## Commands

### Operations
- `reflex watch start` — start watch via reflex

## Usage Examples
- "reflex --help"
- "reflex <args>"

## Installation

```bash
brew install reflex
```

## Examples

```bash
reflex --version
reflex --help
```

- file-watcher\n- development\n- build
