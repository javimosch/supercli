# Osmar Plugin

OpenStreetMap data analyzer and query tool.

## Commands

### Operations
- `osmar data query` — query data via osmar
- `osmar data filter` — filter data via osmar
- `osmar data export` — export data via osmar

## Usage Examples
- "osmar --help"
- "osmar <args>"

## Installation

```bash
go install github.com/codesoap/osmar@latest
```

## Examples

```bash
osmar --version
osmar --help
```
