# Ascii-image-converter Plugin

Convert images into ASCII art.

## Commands

### Operations
- `ascii-image-converter image convert` — convert image via ascii-image-converter

## Usage Examples
- "ascii-image-converter --help"
- "ascii-image-converter <args>"

## Installation

```bash
go install github.com/TheZoraiz/ascii-image-converter@latest
```

## Examples

```bash
ascii-image-converter --version
ascii-image-converter --help
```

## Key Features
- ascii\n- images\n- art
