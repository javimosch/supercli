# Oneshot Plugin

P2P file transfer over WebRTC with NAT traversal.

## Commands

### Operations
- `oneshot file send` — send file via oneshot

## Usage Examples
- "oneshot --help"
- "oneshot <args>"

## Installation

```bash
go install github.com/forestnode-io/oneshot/v2/cmd/oneshot@latest
```

## Examples

```bash
oneshot --version
oneshot --help
```

## Key Features
- file-transfer\n- webrtc\n- p2p
