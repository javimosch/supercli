# Go-gitlab-client Plugin

Command line tool for interacting with GitLab repositories.

## Commands

### Operations
- `go-gitlab-client project list` — list project via glab
- `go-gitlab-client issue list` — list issue via glab
- `go-gitlab-client mr list` — list mr via glab

## Usage Examples
- "go-gitlab-client --help"
- "go-gitlab-client <args>"

## Installation

```bash
go install github.com/plouc/go-gitlab-client@latest
```

## Examples

```bash
glab --version
glab --help
```

## Key Features
- gitlab\n- git\n- cli
