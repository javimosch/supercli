# Fastar Plugin

Fast tar archiver for efficient file compression.

## Commands

### Operations
- `fastar archive create` — create archive via fastar
- `fastar archive extract` — extract archive via fastar
- `fastar archive list` — list archive via fastar
- `fastar archive verify` — verify archive via fastar

## Usage Examples
- "fastar --help"
- "fastar <args>"

## Installation

```bash
cargo install fastar
```

## Examples

```bash
fastar --version
fastar --help
```
