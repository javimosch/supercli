# Goawk Plugin

AWK interpreter written in Go with CSV/TSV support.

## Commands

### Operations
- `goawk text process` — process text via goawk
- `goawk file filter` — filter file via goawk

## Usage Examples
- "goawk --help"
- "goawk <args>"

## Installation

```bash
brew install goawk
```

## Examples

```bash
goawk --version
goawk --help
```

## Key Features
- awk\n- text-processing\n- csv
