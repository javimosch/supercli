# Kube-score Plugin

Kubernetes object analyzer for reliability and security.

## Commands

### Operations
- `kube-score manifest score` — score manifest via kube-score

## Usage Examples
- "kube-score --help"
- "kube-score <args>"

## Installation

```bash
brew install kube-score
```

## Examples

```bash
kube-score --version
kube-score --help
```
