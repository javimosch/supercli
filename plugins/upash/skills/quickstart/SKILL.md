# Upash Plugin

Unified password hashing API for Node.js.

## Commands

### Operations
- `upash password hash` — hash password via upash
- `upash password verify` — verify password via upash

## Usage Examples
- "upash --help"
- "upash <args>"

## Installation

```bash
npm install -g upash-cli
```

## Examples

```bash
upash --version
upash --help
```

## Key Features
- password\n- hashing\n- security
