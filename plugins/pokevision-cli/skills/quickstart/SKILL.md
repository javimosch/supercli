# Pokevision-cli Plugin

CLI tool for Pokemon location tracking.

## Commands

### Operations
- `pokevision-cli location find` — find location via pokevision
- `pokevision-cli pokemon search` — search pokemon via pokevision

## Usage Examples
- "pokevision-cli --help"
- "pokevision-cli <args>"

## Installation

```bash
go install github.com/jmshal/pokevision-cli@latest
```

## Examples

```bash
pokevision --version
pokevision --help
```

## Key Features
- pokemon\n- tracking\n- cli
