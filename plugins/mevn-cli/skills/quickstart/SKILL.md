# Mevn-cli Plugin

CLI tool for the MEVN stack (MongoDB Express Vue Node).

## Commands

### Operations
- `mevn-cli project init` — init project via mevn
- `mevn-cli server create` — create server via mevn
- `mevn-cli component generate` — generate component via mevn

## Usage Examples
- "mevn-cli --help"
- "mevn-cli <args>"

## Installation

```bash
npm install -g mevn-cli
```

## Examples

```bash
mevn --version
mevn --help
```

## Key Features
- vue\n- mongodb\n- full-stack
