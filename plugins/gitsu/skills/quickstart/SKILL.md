# Gitsu Plugin

Switch git user easily.

## Commands

### Operations
- `gitsu user switch` — switch user via gitsu

## Usage Examples
- "gitsu --help"
- "gitsu <args>"

## Installation

```bash
brew install matsuyoshi30/tap/gitsu
```

## Examples

```bash
gitsu --version
gitsu --help
```
