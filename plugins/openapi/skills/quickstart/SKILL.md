# Openapi Plugin

OpenAPI tooling for SDK generation and validation.

## Commands

### Operations
- `openapi spec validate` — validate spec via speakeasy

## Usage Examples
- "openapi --help"
- "openapi <args>"

## Installation

```bash
brew install speakeasy-api/tap/speakeasy
```

## Examples

```bash
speakeasy --version
speakeasy --help
```

## Key Features
- openapi\n- sdk\n- validation
