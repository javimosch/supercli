# Figma-export Plugin

CLI tool to bulk export Figma, FigJam, and Figma Slides files.

## Commands

### Operations
- `figma-export file export` — export file via figma-export
- `figma-export project list` — list project via figma-export
- `figma-export file list` — list file via figma-export
- `figma-export format set` — set format via figma-export
- `figma-export scale set` — set scale via figma-export
- `figma-export directory set` — set directory via figma-export
- `figma-export token set` — set token via figma-export
- `figma-export config show` — show config via figma-export

## Usage Examples
- "figma-export --help"
- "figma-export <args>"

## Installation

```bash
npm install -g figma-export
```
