# Ckube Plugin

Compact Kubernetes dashboard in terminal.

## Commands

### Operations
- `ckube cluster status` — status cluster via ckube
- `ckube pod list` — list pod via ckube
- `ckube deployment list` — list deployment via ckube

## Usage Examples
- "ckube --help"
- "ckube <args>"

## Installation

```bash
go install github.com/CanopyTax/ckube@latest
```

## Examples

```bash
ckube --version
ckube --help
```

## Key Features
- kubernetes\n- dashboard\n- monitoring
