# Mac-cleaner-cli Plugin

macOS CLI to free disk space by clearing caches, logs, Homebrew and Xcode junk.

## Commands

### Operations
- `mac-cleaner-cli cache clear` — clear cache via mac-cleaner
- `mac-cleaner-cli logs clear` — clear logs via mac-cleaner
- `mac-cleaner-cli brew clean` — clean brew via mac-cleaner
- `mac-cleaner-cli xcode clean` — clean xcode via mac-cleaner
- `mac-cleaner-cli all clean` — clean all via mac-cleaner
- `mac-cleaner-cli dry-run run` — run dry-run via mac-cleaner
- `mac-cleaner-cli verbose set` — set verbose via mac-cleaner
- `mac-cleaner-cli config show` — show config via mac-cleaner

## Usage Examples
- "mac-cleaner-cli --help"
- "mac-cleaner-cli <args>"

## Installation

```bash
npm install -g mac-cleaner-cli
```
