# Bpmnlint Plugin

Validate BPMN diagrams from the command line.

## Commands

### Operations
- `bpmnlint file lint` — lint file via bpmnlint

## Usage Examples
- "bpmnlint --help"
- "bpmnlint <args>"

## Installation

```bash
npm install -g bpmnlint
```

## Examples

```bash
bpmnlint --version
bpmnlint --help
```

## Key Features
- bpmn\n- lint\n- validation
