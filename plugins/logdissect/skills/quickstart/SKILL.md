# Logdissect Plugin

CLI utility for analyzing log files and other data with structured log parsing.

## Commands

### Operations
- `logdissect log analyze` — analyze log via logdissect
- `logdissect log filter` — filter log via logdissect
- `logdissect log search` — search log via logdissect
- `logdissect format set` — set format via logdissect
- `logdissect output set` — set output via logdissect
- `logdissect verbose set` — set verbose via logdissect
- `logdissect help show` — show help via logdissect

## Usage Examples
- "logdissect --help"
- "logdissect <args>"

## Installation

```bash
pip install logdissect
```
