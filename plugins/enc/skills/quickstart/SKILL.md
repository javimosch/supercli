# Enc Plugin

Simple encryption and decryption CLI tool.

## Commands

### Operations
- `enc file encrypt` — encrypt file via enc
- `enc file decrypt` — decrypt file via enc

## Usage Examples
- "enc --help"
- "enc <args>"

## Installation

```bash
go install github.com/life4/enc/cmd/enc@latest
```

## Examples

```bash
enc --version
enc --help
```

## Key Features
- encryption\n- security\n- crypto
