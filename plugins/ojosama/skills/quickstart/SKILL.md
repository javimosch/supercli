# Ojosama Plugin

Convert Japanese text to ojou-sama speech style.

## Commands

### Operations
- `ojosama text convert` — convert text via ojosama

## Usage Examples
- "ojosama --help"
- "ojosama <args>"

## Installation

```bash
go install github.com/jiro4989/ojosama@latest
```

## Examples

```bash
ojosama --version
ojosama --help
```

## Key Features
- japanese\n- text-processing\n- converter
