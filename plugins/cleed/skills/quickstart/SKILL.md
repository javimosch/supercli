# Cleed Plugin

Feed reader for the command line.

## Commands

### Operations
- `cleed feed read` — read feed via cleed
- `cleed feed list` — list feed via cleed
- `cleed feed add` — add feed via cleed
- `cleed feed remove` — remove feed via cleed

## Usage Examples
- "cleed --help"
- "cleed <args>"

## Installation

```bash
go install github.com/radulucut/cleed@latest
```

## Examples

```bash
cleed --version
cleed --help
```

- rss\n- feed-reader\n- news
