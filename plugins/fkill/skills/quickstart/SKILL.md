# Fkill Plugin

Fabulously kill processes. Cross-platform.

## Commands

### Operations
- `fkill process kill` — kill process via fkill

## Usage Examples
- "fkill --help"
- "fkill <args>"

## Installation

```bash
npm install -g fkill-cli
```

## Examples

```bash
fkill --version
fkill --help
```

## Key Features
- process\n- kill\n- system
