# Rts Plugin

Mock JSON server and REST API testing tool.

## Commands

### Operations
- `rts server start` — start server via rts

## Usage Examples
- "rts --help"
- "rts <args>"

## Installation

```bash
go install github.com/galeone/rts@latest
```

## Examples

```bash
rts --version
rts --help
```

- json\n- rest\n- api\n- mock
