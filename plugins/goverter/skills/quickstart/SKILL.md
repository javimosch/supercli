# Goverter Plugin

Type-safe Go converter generator.

## Commands

### Operations
- `goverter types generate` — generate types via goverter

## Usage Examples
- "goverter --help"
- "goverter <args>"

## Installation

```bash
go install github.com/jmattheis/goverter/cmd/goverter@latest
```

## Examples

```bash
goverter --version
goverter --help
```

## Key Features
- go\n- converter\n- code-generation
