# React-native-version Plugin

Automate React Native versioning.

## Commands

### Operations
- `react-native-version version bump` — bump version via react-native-version

## Usage Examples
- "react-native-version --help"
- "react-native-version <args>"

## Installation

```bash
npm install -g react-native-version
```

## Examples

```bash
react-native-version --version
react-native-version --help
```

- react-native\n- versioning\n- mobile
