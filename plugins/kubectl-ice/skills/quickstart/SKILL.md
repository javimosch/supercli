# Kubectl-ice Plugin

Kubernetes resource usage monitor and optimizer for pods and containers.

## Commands

### Operations
- `kubectl-ice pod usage` — usage pod via kubectl-ice
- `kubectl-ice container usage` — usage container via kubectl-ice
- `kubectl-ice node usage` — usage node via kubectl-ice
- `kubectl-ice quota show` — show quota via kubectl-ice
- `kubectl-ice limits show` — show limits via kubectl-ice
- `kubectl-ice requests show` — show requests via kubectl-ice
- `kubectl-ice restarts show` — show restarts via kubectl-ice
- `kubectl-ice splits show` — show splits via kubectl-ice
- `kubectl-ice lifecycle show` — show lifecycle via kubectl-ice
- `kubectl-ice probes show` — show probes via kubectl-ice
- `kubectl-ice images show` — show images via kubectl-ice
- `kubectl-ice volumes show` — show volumes via kubectl-ice
- `kubectl-ice network show` — show network via kubectl-ice
- `kubectl-ice ip show` — show ip via kubectl-ice
- `kubectl-ice ports show` — show ports via kubectl-ice
- `kubectl-ice node-ports show` — show node-ports via kubectl-ice
- `kubectl-ice tolerations show` — show tolerations via kubectl-ice
- `kubectl-ice affinities show` — show affinities via kubectl-ice
- `kubectl-ice rbac show` — show rbac via kubectl-ice

## Usage Examples
