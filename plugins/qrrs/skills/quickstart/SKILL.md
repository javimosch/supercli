# Qrrs Plugin

CLI QR code generator and reader written in Rust.

## Commands

### Operations
- `qrrs qr generate` — generate qr via qrrs
- `qrrs qr read` — read qr via qrrs
- `qrrs text encode` — encode text via qrrs
- `qrrs image decode` — decode image via qrrs

## Usage Examples
- "qrrs --help"
- "qrrs <args>"

## Installation

```bash
cargo install qrrs
```

## Examples

```bash
qrrs --version
qrrs --help
```
