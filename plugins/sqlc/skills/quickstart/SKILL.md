# Sqlc Plugin

Generate type-safe Go code from SQL.

## Commands

### Operations
- `sqlc sql generate` — generate sql via sqlc

## Usage Examples
- "sqlc --help"
- "sqlc <args>"

## Installation

```bash
brew install sqlc
```

## Examples

```bash
sqlc --version
sqlc --help
```

## Key Features
- sql\n- code-generation\n- database
