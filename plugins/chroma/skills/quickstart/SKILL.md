# Chroma Plugin

A general-purpose syntax highlighter in pure Go.

## Commands

### Operations
- `chroma file highlight` — highlight file via chroma

## Usage Examples
- "chroma --help"
- "chroma <args>"

## Installation

```bash
go install github.com/alecthomas/chroma/cmd/chroma@latest
```

## Examples

```bash
chroma --version
chroma --help
```

## Key Features
- syntax-highlighting\n- text\n- formatting
