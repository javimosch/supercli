# Polaris Plugin

Validate Kubernetes best practices in CI/CD.

## Commands

### Operations
- `polaris cluster audit` — audit cluster via polaris
- `polaris manifest audit` — audit manifest via polaris

## Usage Examples
- "polaris --help"
- "polaris <args>"

## Installation

```bash
brew install FairwindsOps/tap/polaris
```

## Examples

```bash
polaris --version
polaris --help
```

## Key Features
- kubernetes\n- best-practices\n- audit
