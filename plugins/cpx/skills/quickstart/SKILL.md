# Cpx Plugin

Copy files with watch mode and glob support.

## Commands

### Operations
- `cpx files copy` — copy files via cpx

## Usage Examples
- "cpx --help"
- "cpx <args>"

## Installation

```bash
npm install -g cpx
```

## Examples

```bash
cpx --version
cpx --help
```

## Key Features
- copy\n- files\n- build
