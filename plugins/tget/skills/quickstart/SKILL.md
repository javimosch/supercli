# Tget Plugin

Torrent file downloader and getter CLI.

## Commands

### Operations
- `tget torrent download` — download torrent via tget

## Usage Examples
- "tget --help"
- "tget <args>"

## Installation

```bash
npm install -g tget
```

## Examples

```bash
tget --version
tget --help
```

## Key Features
- torrent\n- download\n- p2p
