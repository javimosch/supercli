# Craftql Plugin

CLI tool to visualize GraphQL schemas and output graph data structures.

## Commands

### Operations
- `craftql schema visualize` — visualize schema via craftql
- `craftql schema output` — output schema via craftql
- `craftql schema dot` — dot schema via craftql

## Usage Examples
- "craftql --help"
- "craftql <args>"

## Installation

```bash
cargo install craftql
```

## Examples

```bash
craftql --version
craftql --help
```

- graphql\n- visualization\n- rust
