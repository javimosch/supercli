# Sonnerie Plugin

Simple timeseries database with CLI interface.

## Commands

### Operations
- `sonnerie data insert` — insert data via sonnerie
- `sonnerie data query` — query data via sonnerie
- `sonnerie database create` — create database via sonnerie
- `sonnerie database compact` — compact database via sonnerie
- `sonnerie database info` — info database via sonnerie
- `sonnerie schema define` — define schema via sonnerie
- `sonnerie export run` — run export via sonnerie
- `sonnerie import run` — run import via sonnerie
- `sonnerie config show` — show config via sonnerie

## Usage Examples
- "sonnerie --help"
- "sonnerie <args>"

## Installation

```bash
cargo install sonnerie
```
