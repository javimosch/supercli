# Conftest Plugin

Write tests against structured configuration data using OPA.

## Commands

### Operations
- `conftest config test` — test config via conftest
- `conftest config verify` — verify config via conftest

## Usage Examples
- "conftest --help"
- "conftest <args>"

## Installation

```bash
brew install conftest
```

## Examples

```bash
conftest --version
conftest --help
```

## Key Features
- opa\n- policy\n- testing
