# Soundcloud-dl Plugin

Download tracks and playlists from SoundCloud.

## Commands

### Operations
- `soundcloud-dl track download` — download track via soundcloud-dl
- `soundcloud-dl playlist download` — download playlist via soundcloud-dl

## Usage Examples
- "soundcloud-dl --help"
- "soundcloud-dl <args>"

## Installation

```bash
go install github.com/AYehia0/soundcloud-dl@latest
```

## Examples

```bash
soundcloud-dl --version
soundcloud-dl --help
```

## Key Features
- soundcloud\n- download\n- music
