# Kubens Plugin

Faster way to switch between Kubernetes namespaces.

## Commands

### Operations
- `kubens namespace switch` — switch namespace via kubens
- `kubens namespace list` — list namespace via kubens

## Usage Examples
- "kubens --help"
- "kubens <args>"

## Installation

```bash
brew install kubectx
```

## Examples

```bash
kubens --version
kubens --help
```

## Key Features
- kubernetes\n- kubectl\n- namespace
