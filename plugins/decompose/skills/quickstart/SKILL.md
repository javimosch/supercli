# Decompose Plugin

Docker dependency graph generator and analyzer.

## Commands

### Operations
- `decompose graph generate` — generate graph via decompose

## Usage Examples
- "decompose --help"
- "decompose <args>"

## Installation

```bash
go install github.com/s0rg/decompose/cmd/decompose@latest
```

## Examples

```bash
decompose --version
decompose --help
```

## Key Features
