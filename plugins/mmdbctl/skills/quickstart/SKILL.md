# Mmdbctl Plugin

MMDB file management CLI supporting various operations on MMDB database files.

## Commands

### Operations
- `mmdbctl db read` — read db via mmdbctl
- `mmdbctl db export` — export db via mmdbctl
- `mmdbctl db verify` — verify db via mmdbctl

## Usage Examples
- "mmdbctl --help"
- "mmdbctl <args>"

## Installation

```bash
brew install ipinfo/tap/mmdbctl
```

## Examples

```bash
mmdbctl --version
mmdbctl --help
```

## Key Features
- mmdb\n- database\n- network
