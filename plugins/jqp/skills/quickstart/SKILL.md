# Jqp Plugin

TUI JSON processor and interactive jq playground.

## Commands

### Operations
- `jqp json process` — process json via jqp

## Usage Examples
- "jqp --help"
- "jqp <args>"

## Installation

```bash
brew install jqp
```

## Examples

```bash
jqp --version
jqp --help
```

## Key Features
- json\n- jq\n- tui
