# Wtp Plugin

Weather tool with pretty terminal output.

## Commands

### Operations
- `wtp weather get` — get weather via wtp

## Usage Examples
- "wtp --help"
- "wtp <args>"

## Installation

```bash
go install github.com/satococoa/wtp@latest
```

## Examples

```bash
wtp --version
wtp --help
```

## Key Features
- weather\n- terminal\n- forecast
