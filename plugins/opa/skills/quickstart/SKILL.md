# Opa Plugin

Open Policy Agent command line for policy evaluation.

## Commands

### Operations
- `opa policy eval` — eval policy via opa
- `opa policy test` — test policy via opa
- `opa policy build` — build policy via opa

## Usage Examples
- "opa --help"
- "opa <args>"

## Installation

```bash
brew install opa
```

## Examples

```bash
opa --version
opa --help
```

## Key Features
- policy\n- opa\n- rego
