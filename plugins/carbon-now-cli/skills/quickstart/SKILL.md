# Carbon-now-cli Plugin

Beautiful images of your code from the command line.

## Commands

### Operations
- `carbon-now-cli file convert` — convert file via carbon-now

## Usage Examples
- "carbon-now-cli --help"
- "carbon-now-cli <args>"

## Installation

```bash
npm install -g carbon-now-cli
```

## Examples

```bash
carbon-now --version
carbon-now --help
```

## Key Features
- code\n- images\n- sharing
