# Wol Plugin

Wake up your devices with a single command. A Wake-On-LAN tool.

## Commands

### Operations
- `wol device wake` — wake device via wol
- `wol device list` — list device via wol

## Usage Examples
- "wol --help"
- "wol <args>"

## Installation

```bash
go install github.com/Trugamr/wol@latest
```

## Examples

```bash
wol --version
wol --help
```

## Key Features
- network\n- wake-on-lan\n- iot
