# Cooklang-chef Plugin

CLI to manage and organize Cooklang recipes.

## Commands

### Operations
- `cooklang-chef recipe list` — list recipe via chef
- `cooklang-chef recipe show` — show recipe via chef
- `cooklang-chef recipe add` — add recipe via chef
- `cooklang-chef recipe remove` — remove recipe via chef
- `cooklang-chef recipe scale` — scale recipe via chef
- `cooklang-chef recipe convert` — convert recipe via chef
- `cooklang-chef ingredient list` — list ingredient via chef
- `cooklang-chef shopping list` — list shopping via chef

## Usage Examples
- "cooklang-chef --help"
- "cooklang-chef <args>"

## Installation

```bash
cargo install cooklang-chef
```

## Examples

```bash
chef --version
chef --help
```
