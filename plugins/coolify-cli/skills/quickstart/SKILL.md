# Coolify-cli Plugin

Self-hostable Heroku/Netlify alternative CLI.

## Commands

### Operations
- `coolify-cli project deploy` — deploy project via coolify
- `coolify-cli project init` — init project via coolify
- `coolify-cli server setup` — setup server via coolify

## Usage Examples
- "coolify-cli --help"
- "coolify-cli <args>"

## Installation

```bash
npm install -g coolify-cli
```

## Examples

```bash
coolify --version
coolify --help
```
