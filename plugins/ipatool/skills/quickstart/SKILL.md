# Ipatool Plugin

Command line tool for searching and downloading iOS apps.

## Commands

### Operations
- `ipatool app search` — search app via ipatool
- `ipatool app download` — download app via ipatool

## Usage Examples
- "ipatool --help"
- "ipatool <args>"

## Installation

```bash
brew install ipatool
```

## Examples

```bash
ipatool --version
ipatool --help
```
