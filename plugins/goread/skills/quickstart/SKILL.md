# Goread Plugin

Command line ePub reader for Go.

## Commands

### Operations
- `goread book read` — read book via goread

## Usage Examples
- "goread --help"
- "goread <args>"

## Installation

```bash
go install github.com/Ray-D-Song/goread@latest
```

## Examples

```bash
goread --version
goread --help
```

## Key Features
- epub\n- reader\n- ebook
