# Run Plugin

Run commands in the languages you love.

## Commands

### Operations
- `run script execute` — execute script via run
- `run script list` — list script via run
- `run language detect` — detect language via run
- `run config show` — show config via run

## Usage Examples
- "run --help"
- "run <args>"

## Installation

```bash
cargo install run
```

## Examples

```bash
run --version
run --help
```

## Key Features
- scripting\n- multi-language\n- rust
