# Domain-cli Plugin

CLI tool for WHOIS and domain information lookup.

## Commands

### Operations
- `domain-cli domain lookup` — lookup domain via domain

## Usage Examples
- "domain-cli --help"
- "domain-cli <args>"

## Installation

```bash
npm install -g domain-cli
```

## Examples

```bash
domain --version
domain --help
```

## Key Features
- domain\n- whois\n- networking
