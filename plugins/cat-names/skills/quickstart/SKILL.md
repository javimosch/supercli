# Cat-names Plugin

Get popular cat names from the command line.

## Commands

### Operations
- `cat-names name random` — random name via cat-names

## Usage Examples
- "cat-names --help"
- "cat-names <args>"

## Installation

```bash
npm install -g cat-names
```

## Examples

```bash
cat-names --version
cat-names --help
```
