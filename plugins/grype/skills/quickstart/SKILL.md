# Grype Plugin

Vulnerability scanner for container images and filesystems.

## Commands

### Operations
- `grype image scan` — scan image via grype
- `grype dir scan` — scan dir via grype
- `grype sbom scan` — scan sbom via grype

## Usage Examples
- "grype --help"
- "grype <args>"

## Installation

```bash
brew install grype
```

## Examples

```bash
grype --version
grype --help
```
