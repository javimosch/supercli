# Parrotsay Plugin

ASCII art parrot generator for the terminal.

## Commands

### Operations
- `parrotsay text display` — display text via parrotsay

## Usage Examples
- "parrotsay --help"
- "parrotsay <args>"

## Installation

```bash
npm install -g parrotsay
```

## Examples

```bash
parrotsay --version
parrotsay --help
```

## Key Features
- ascii\n- art\n- fun
