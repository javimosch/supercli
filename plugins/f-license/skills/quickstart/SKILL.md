# F-license Plugin

License checker and validator for open-source projects.

## Commands

### Operations
- `f-license license check` — check license via f-license

## Usage Examples
- "f-license --help"
- "f-license <args>"

## Installation

```bash
go install github.com/furkansenharputlu/f-license@latest
```

## Examples

```bash
f-license --version
f-license --help
```
