# Umoci Plugin

Umoci modifies Open Container images without needing superuser privileges.

## Commands

### Operations
- `umoci image unpack` — unpack image via umoci
- `umoci image repack` — repack image via umoci
- `umoci image stat` — stat image via umoci
- `umoci image insert` — insert image via umoci

## Usage Examples
- "umoci --help"
- "umoci <args>"

## Installation

```bash
brew install umoci
```

## Examples

```bash
umoci --version
umoci --help
```
