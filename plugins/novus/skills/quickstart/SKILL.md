# Novus Plugin

Blazingly fast and efficient package manager for Windows.

## Commands

### Operations
- `novus package install` — install package via novus
- `novus package remove` — remove package via novus
- `novus package update` — update package via novus
- `novus package list` — list package via novus
- `novus package search` — search package via novus
- `novus package info` — info package via novus
- `novus cache clean` — clean cache via novus
- `novus config show` — show config via novus
- `novus remote list` — list remote via novus
- `novus remote add` — add remote via novus
- `novus remote remove` — remove remote via novus
- `novus remote update` — update remote via novus
- `novus verify run` — run verify via novus

## Usage Examples
- "novus --help"
- "novus <args>"
