# Go-perftuner Plugin

Performance tuning helper for Go programs.

## Commands

### Operations
- `go-perftuner code check` — check code via go-perftuner

## Usage Examples
- "go-perftuner --help"
- "go-perftuner <args>"

## Installation

```bash
go install github.com/go-perf/go-perftuner/cmd/go-perftuner@latest
```

## Examples

```bash
go-perftuner --version
go-perftuner --help
```

## Key Features
- performance\n- benchmark\n- go
