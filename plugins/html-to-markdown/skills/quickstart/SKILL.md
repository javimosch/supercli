# Html-to-markdown Plugin

Convert HTML to Markdown from the command line.

## Commands

### Operations
- `html-to-markdown file convert` — convert file via html-to-markdown

## Usage Examples
- "html-to-markdown --help"
- "html-to-markdown <args>"

## Installation

```bash
go install github.com/JohannesKaufmann/html-to-markdown/cmd/html-to-markdown@latest
```

## Examples

```bash
html-to-markdown --version
html-to-markdown --help
```

## Key Features
- html\n- markdown\n- converter
