# Node-rename-cli Plugin

Batch rename files and directories.

## Commands

### Operations
- `node-rename-cli files rename` — rename files via rename

## Usage Examples
- "node-rename-cli --help"
- "node-rename-cli <args>"

## Installation

```bash
npm install -g rename-cli
```

## Examples

```bash
rename --version
rename --help
```

## Key Features
- rename\n- files\n- batch
