# Github-files-fetcher Plugin

Download files and directories from GitHub.

## Commands

### Operations
- `github-files-fetcher repo fetch` — fetch repo via github-files-fetcher

## Usage Examples
- "github-files-fetcher --help"
- "github-files-fetcher <args>"

## Installation

```bash
npm install -g github-files-fetcher
```

## Examples

```bash
github-files-fetcher --version
github-files-fetcher --help
```

## Key Features
- github\n- download\n- files
