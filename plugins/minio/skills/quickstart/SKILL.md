# Minio Plugin

High performance object storage with S3 compatible API.

## Commands

### Operations
- `minio bucket ls` — ls bucket via mc
- `minio bucket mb` — mb bucket via mc
- `minio object cp` — cp object via mc
- `minio object mirror` — mirror object via mc

## Usage Examples
- "minio --help"
- "minio <args>"

## Installation

```bash
brew install minio/stable/mc
```

## Examples

```bash
mc --version
mc --help
```
