# Static-react Plugin

Zero-config static site generator for React.

## Commands

### Operations
- `static-react site build` — build site via static-react

## Usage Examples
- "static-react --help"
- "static-react <args>"

## Installation

```bash
npm install -g static-react
```

## Examples

```bash
static-react --version
static-react --help
```

## Key Features
- react\n- static-site\n- ssg
