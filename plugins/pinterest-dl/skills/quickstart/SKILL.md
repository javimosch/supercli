# Pinterest-dl Plugin

Unofficial Pinterest media downloader for downloading images and videos.

## Commands

### Operations
- `pinterest-dl media download` — download media via pinterest-dl
- `pinterest-dl board list` — list board via pinterest-dl
- `pinterest-dl pin save` — save pin via pinterest-dl
- `pinterest-dl format set` — set format via pinterest-dl
- `pinterest-dl output set` — set output via pinterest-dl
- `pinterest-dl verbose set` — set verbose via pinterest-dl
- `pinterest-dl help show` — show help via pinterest-dl

## Usage Examples
- "pinterest-dl --help"
- "pinterest-dl <args>"

## Installation

```bash
pip install pinterest-dl
```

## Examples

```bash
pinterest-dl --version
pinterest-dl --help
```
