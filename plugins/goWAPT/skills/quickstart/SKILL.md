# GoWAPT Plugin

Web application vulnerability scanner and fuzzer.

## Commands

### Operations
- `goWAPT target scan` — scan target via goWAPT

## Usage Examples
- "goWAPT --help"
- "goWAPT <args>"

## Installation

```bash
go install github.com/dzonerzy/goWAPT@latest
```

## Examples

```bash
goWAPT --version
goWAPT --help
```

## Key Features
- security\n- scanner\n- fuzzer
