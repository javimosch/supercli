# Protocurl Plugin

cURL for Protobuf: interact with Protobuf over HTTP REST endpoints.

## Commands

### Operations
- `protocurl request send` — send request via protocurl
- `protocurl proto describe` — describe proto via protocurl
- `protocurl config show` — show config via protocurl

## Usage Examples
- "protocurl --help"
- "protocurl <args>"

## Installation

```bash
brew install qaware/tap/protocurl
```

## Examples

```bash
protocurl --version
protocurl --help
```

## Key Features
- protobuf\n- http\n- api-testing
