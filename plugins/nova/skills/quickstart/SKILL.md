# Nova Plugin

Finds outdated Helm charts running in a cluster.

## Commands

### Operations
- `nova charts find` — find charts via nova

## Usage Examples
- "nova --help"
- "nova <args>"

## Installation

```bash
brew install FairwindsOps/tap/nova
```

## Examples

```bash
nova --version
nova --help
```

## Key Features
- helm\n- kubernetes\n- charts
