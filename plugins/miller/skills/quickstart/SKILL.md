# Miller Plugin

Like awk/sed/join/sort for name-indexed data such as CSV, TSV, JSON.

## Commands

### Operations
- `miller data process` — process data via mlr

## Usage Examples
- "miller --help"
- "miller <args>"

## Installation

```bash
brew install miller
```

## Examples

```bash
mlr --version
mlr --help
```
