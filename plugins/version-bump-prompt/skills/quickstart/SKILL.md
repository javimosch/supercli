# Version-bump-prompt Plugin

Interactive version bumping with prompts.

## Commands

### Operations
- `version-bump-prompt version bump` — bump version via version-bump-prompt

## Usage Examples
- "version-bump-prompt --help"
- "version-bump-prompt <args>"

## Installation

```bash
npm install -g version-bump-prompt
```

## Examples

```bash
version-bump-prompt --version
version-bump-prompt --help
```

## Key Features
- versioning\n- semver\n- prompt
