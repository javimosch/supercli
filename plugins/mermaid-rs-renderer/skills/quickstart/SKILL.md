# Mermaid-rs-renderer Plugin

Fast native Rust Mermaid diagram renderer without browser.

## Commands

### Operations
- `mermaid-rs-renderer diagram render` — render diagram via mermaid-rs
- `mermaid-rs-renderer file convert` — convert file via mermaid-rs
- `mermaid-rs-renderer input process` — process input via mermaid-rs
- `mermaid-rs-renderer output save` — save output via mermaid-rs
- `mermaid-rs-renderer format set` — set format via mermaid-rs
- `mermaid-rs-renderer theme set` — set theme via mermaid-rs
- `mermaid-rs-renderer config show` — show config via mermaid-rs

## Usage Examples
- "mermaid-rs-renderer --help"
- "mermaid-rs-renderer <args>"

## Installation

```bash
cargo install mermaid-rs-renderer
```

