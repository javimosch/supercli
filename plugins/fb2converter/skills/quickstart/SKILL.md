# Fb2converter Plugin

Convert FB2 e-book files to other formats.

## Commands

### Operations
- `fb2converter file convert` — convert file via fb2converter

## Usage Examples
- "fb2converter --help"
- "fb2converter <args>"

## Installation

```bash
go install github.com/rupor-github/fb2converter@latest
```

## Examples

```bash
fb2converter --version
fb2converter --help
```

## Key Features
- ebook\n- fb2\n- converter
