# Dosxvpn Plugin

Simple macOS VPN client for DigitalOcean Droplets.

## Commands

### Operations
- `dosxvpn vpn connect` — connect vpn via dosxvpn
- `dosxvpn vpn disconnect` — disconnect vpn via dosxvpn
- `dosxvpn vpn status` — status vpn via dosxvpn
- `dosxvpn server list` — list server via dosxvpn

## Usage Examples
- "dosxvpn --help"
- "dosxvpn <args>"

## Installation

```bash
go install github.com/dan-v/dosxvpn@latest
```

## Examples

```bash
dosxvpn --version
dosxvpn --help
```
