# Ftl Plugin

Faster than light deployment tool.

## Commands

### Operations
- `ftl deploy run` — run deploy via ftl
- `ftl build run` — run build via ftl

## Usage Examples
- "ftl --help"
- "ftl <args>"

## Installation

```bash
brew install yarlson/tap/ftl
```

## Examples

```bash
ftl --version
ftl --help
```

## Key Features
- deployment\n- devops\n- ssh
