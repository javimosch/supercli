# Tfvar Plugin

Generate Terraform variable definitions from existing code.

## Commands

### Operations
- `tfvar config generate` — generate config via tfvar

## Usage Examples
- "tfvar --help"
- "tfvar <args>"

## Installation

```bash
go install github.com/shihanng/tfvar@latest
```

## Examples

```bash
tfvar --version
tfvar --help
```

## Key Features
- terraform\n- variables\n- iac
