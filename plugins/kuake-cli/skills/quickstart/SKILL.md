# Kuake-cli Plugin

Quick command launcher and alias manager.

## Commands

### Operations
- `kuake-cli command run` — run command via kuake
- `kuake-cli alias list` — list alias via kuake

## Usage Examples
- "kuake-cli --help"
- "kuake-cli <args>"

## Installation

```bash
go install github.com/zhangjingwei/kuake_cli@latest
```

## Examples

```bash
kuake --version
kuake --help
```

- launcher\n- aliases\n- productivity
