# Bt2qbt Plugin

Migrate BitTorrent to qBittorrent torrent files.

## Commands

### Operations
- `bt2qbt torrent migrate` — migrate torrent via bt2qbt

## Usage Examples
- "bt2qbt --help"
- "bt2qbt <args>"

## Installation

```bash
go install github.com/rumanzo/bt2qbt@latest
```

## Examples

```bash
bt2qbt --version
bt2qbt --help
```

## Key Features
- bittorrent\n- qbittorrent\n- migration
