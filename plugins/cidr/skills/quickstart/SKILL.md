# Cidr Plugin

CIDR range calculator and IP subnet utility.

## Commands

### Operations
- `cidr ip calculate` — calculate ip via cidr

## Usage Examples
- "cidr --help"
- "cidr <args>"

## Installation

```bash
brew install cidr
```

## Examples

```bash
cidr --version
cidr --help
```

## Key Features
- network\n- cidr\n- subnet
