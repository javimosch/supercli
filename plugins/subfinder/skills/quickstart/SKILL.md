# Subfinder Plugin

Fast passive subdomain enumeration tool.

## Commands

### Operations
- `subfinder domain discover` — discover domain via subfinder
- `subfinder domain list` — list domain via subfinder

## Usage Examples
- "subfinder --help"
- "subfinder <args>"

## Installation

```bash
go install github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest
```

## Examples

```bash
subfinder --version
subfinder --help
```

- security\n- reconnaissance\n- subdomain
