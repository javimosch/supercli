# Scmpuff Plugin

Numeric file selection shortcuts for common git commands.

## Commands

### Operations
- `scmpuff status show` — show status via scmpuff
- `scmpuff cmd exec` — exec cmd via scmpuff

## Usage Examples
- "scmpuff --help"
- "scmpuff <args>"

## Installation

```bash
brew install scmpuff
```

## Examples

```bash
scmpuff --version
scmpuff --help
```
