# Localias Plugin

Manage local development aliases and hostnames.

## Commands

### Operations
- `localias alias list` — list alias via localias
- `localias alias add` — add alias via localias

## Usage Examples
- "localias --help"
- "localias <args>"

## Installation

```bash
go install github.com/peterldowns/localias@latest
```

## Examples

```bash
localias --version
localias --help
```

## Key Features
- aliases\n- hosts\n- development
