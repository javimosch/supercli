# Rvc-cli Plugin

RVC and UVR tools for voice cloning easily and free.

## Commands

### Operations
- `rvc-cli voice clone` — clone voice via rvc-cli
- `rvc-cli voice train` — train voice via rvc-cli
- `rvc-cli voice infer` — infer voice via rvc-cli
- `rvc-cli model list` — list model via rvc-cli
- `rvc-cli output set` — set output via rvc-cli
- `rvc-cli config show` — show config via rvc-cli
- `rvc-cli verbose set` — set verbose via rvc-cli
- `rvc-cli help show` — show help via rvc-cli

## Usage Examples
- "rvc-cli --help"
- "rvc-cli <args>"

pip install rvc-cli
