# Databooks Plugin

CLI tool to reduce friction between data scientists by removing notebook metadata and resolving git conflicts.

## Commands

### Operations
- `databooks notebook clean` — clean notebook via databooks
- `databooks metadata remove` — remove metadata via databooks
- `databooks conflict resolve` — resolve conflict via databooks
- `databooks git sync` — sync git via databooks
- `databooks config show` — show config via databooks
- `databooks verbose set` — set verbose via databooks
- `databooks help show` — show help via databooks

## Usage Examples
- "databooks --help"
- "databooks <args>"

## Installation

```bash
pip install databooks
```

## Examples

```bash
databooks --version
```
