# Presto Plugin

Lightweight tunnel and proxy CLI tool.

## Commands

### Operations
- `presto tunnel create` — create tunnel via presto

## Usage Examples
- "presto --help"
- "presto <args>"

## Installation

```bash
go install github.com/paramientos/presto@latest
```

## Examples

```bash
presto --version
presto --help
```

## Key Features
- tunnel\n- proxy\n- networking
