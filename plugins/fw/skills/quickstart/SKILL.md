# Fw Plugin

Workspace productivity booster for managing multiple projects.

## Commands

### Operations
- `fw project list` — list project via fw
- `fw project add` — add project via fw
- `fw project remove` — remove project via fw
- `fw project update` — update project via fw
- `fw workspace setup` — setup workspace via fw
- `fw config show` — show config via fw

## Usage Examples
- "fw --help"
- "fw <args>"

## Installation

```bash
cargo install fw
```

## Examples

```bash
fw --version
fw --help
```
