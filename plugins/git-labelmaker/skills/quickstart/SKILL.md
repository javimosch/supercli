# Git-labelmaker Plugin

CLI tool for creating and editing GitHub labels.

## Commands

### Operations
- `git-labelmaker label create` — create label via git-labelmaker
- `git-labelmaker label edit` — edit label via git-labelmaker

## Usage Examples
- "git-labelmaker --help"
- "git-labelmaker <args>"

## Installation

```bash
npm install -g git-labelmaker
```

## Examples

```bash
git-labelmaker --version
git-labelmaker --help
```

## Key Features
- github\n- labels\n- git
