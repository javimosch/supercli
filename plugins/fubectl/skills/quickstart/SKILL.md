# Fubectl Plugin

Reduced kubectl interactions wrapper.

## Commands

### Operations
- `fubectl resource get` — get resource via fubectl
- `fubectl resource describe` — describe resource via fubectl

## Usage Examples
- "fubectl --help"
- "fubectl <args>"

## Installation

```bash
brew install fubectl
```

## Examples

```bash
fubectl --version
fubectl --help
```
