# Create-component-app Plugin

Generate React components from the command line.

## Commands

### Operations
- `create-component-app component generate` — generate component via create-component-app

## Usage Examples
- "create-component-app --help"
- "create-component-app <args>"

## Installation

```bash
npm install -g create-component-app
```

## Examples

```bash
create-component-app --version
create-component-app --help
```

## Key Features
- react\n- components\n- generator
