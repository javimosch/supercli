# Cdk-notifier Plugin

AWS CDK deployment notifier for Slack and other channels.

## Commands

### Operations
- `cdk-notifier notify send` — send notify via cdk-notifier
- `cdk-notifier config show` — show config via cdk-notifier

## Usage Examples
- "cdk-notifier --help"
- "cdk-notifier <args>"

## Installation

```bash
go install github.com/karlderkaefer/cdk-notifier@latest
```

## Examples

```bash
cdk-notifier --version
cdk-notifier --help
```

## Key Features
- aws-cdk\n- deployment\n- notifications
