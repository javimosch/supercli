# Gk Plugin

Go-Kit generator for creating Go microservices.

## Commands

### Operations
- `gk service generate` — generate service via gk
- `gk endpoint generate` — generate endpoint via gk
- `gk middleware generate` — generate middleware via gk

## Usage Examples
- "gk --help"
- "gk <args>"

## Installation

```bash
go install github.com/kujtimiihoxha/gk@latest
```

## Examples

```bash
gk --version
gk --help
```

## Key Features
- go-kit\n- microservices\n- generator
