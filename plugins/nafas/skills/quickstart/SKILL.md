# Nafas Plugin

Breathing Gymnastics Application for guided breathing exercises.

## Commands

### Operations
- `nafas exercise start` — start exercise via nafas
- `nafas exercise list` — list exercise via nafas
- `nafas duration set` — set duration via nafas
- `nafas pattern set` — set pattern via nafas
- `nafas verbose set` — set verbose via nafas
- `nafas help show` — show help via nafas

## Usage Examples
- "nafas --help"
- "nafas <args>"

pip install nafas
