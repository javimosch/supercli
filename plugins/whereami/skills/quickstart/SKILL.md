# Whereami Plugin

Get your geolocation information from the command line.

## Commands

### Operations
- `whereami location get` — get location via whereami

## Usage Examples
- "whereami --help"
- "whereami <args>"

## Installation

```bash
npm install -g @rafaelrinaldi/whereami
```

## Examples

```bash
whereami --version
whereami --help
```

## Key Features
- geolocation\n- ip\n- location
