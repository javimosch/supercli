# Writeas-cli Plugin

Command line publishing tool for Write.as.

## Commands

### Operations
- `writeas-cli post publish` — publish post via writeas
- `writeas-cli post list` — list post via writeas
- `writeas-cli post delete` — delete post via writeas

## Usage Examples
- "writeas-cli --help"
- "writeas-cli <args>"

## Installation

```bash
brew install writeas
```

## Examples

```bash
writeas --version
writeas --help
```

## Key Features
