# Ds1054z Plugin

Python package for the Rigol DS1054Z Oscilloscope with CLI interface.

## Commands

### Operations
- `ds1054z device connect` — connect device via ds1054z
- `ds1054z waveform capture` — capture waveform via ds1054z
- `ds1054z waveform save` — save waveform via ds1054z
- `ds1054z config show` — show config via ds1054z
- `ds1054z output set` — set output via ds1054z
- `ds1054z verbose set` — set verbose via ds1054z
- `ds1054z help show` — show help via ds1054z

## Usage Examples
- "ds1054z --help"
- "ds1054z <args>"
