# Imagecli Plugin

Command line image processing tool written in Rust.

## Commands

### Operations
- `imagecli image process` — process image via imagecli
- `imagecli filter apply` — apply filter via imagecli
- `imagecli resize run` — run resize via imagecli
- `imagecli rotate run` — run rotate via imagecli
- `imagecli crop run` — run crop via imagecli
- `imagecli format convert` — convert format via imagecli
- `imagecli pipeline run` — run pipeline via imagecli
- `imagecli show image` — image show via imagecli
- `imagecli info image` — image info via imagecli

## Usage Examples
- "imagecli --help"
- "imagecli <args>"

## Installation

```bash
cargo install imagecli
```

## Examples

```bash
imagecli --version
imagecli --help
```
