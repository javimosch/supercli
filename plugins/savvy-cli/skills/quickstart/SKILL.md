# Savvy-cli Plugin

Record, replay and share terminal sessions.

## Commands

### Operations
- `savvy-cli session record` — record session via savvy
- `savvy-cli session replay` — replay session via savvy
- `savvy-cli session share` — share session via savvy

## Usage Examples
- "savvy-cli --help"
- "savvy-cli <args>"

## Installation

```bash
brew install getsavvy/tap/savvy
```

## Examples

```bash
savvy --version
savvy --help
```

## Key Features
- terminal\n- recording\n- replay
