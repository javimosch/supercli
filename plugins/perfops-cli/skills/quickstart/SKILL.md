# Perfops-cli Plugin

Command line tool to interact with servers around the world.

## Commands

### Operations
- `perfops-cli ping run` — run ping via perfops
- `perfops-cli dns lookup` — lookup dns via perfops
- `perfops-cli mtr run` — run mtr via perfops
- `perfops-cli traceroute run` — run traceroute via perfops
- `perfops-cli curl run` — run curl via perfops
- `perfops-cli latency test` — test latency via perfops
- `perfops-cli cdn compare` — compare cdn via perfops

## Usage Examples
- "perfops-cli --help"
- "perfops-cli <args>"

## Installation

```bash
go install github.com/ProspectOne/perfops-cli@latest
```

## Examples

```bash
perfops --version
perfops --help
```
