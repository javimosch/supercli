# Markpdf Plugin

Watermark PDF files using image or text overlays.

## Commands

### Operations
- `markpdf pdf watermark` — watermark pdf via markpdf
- `markpdf text stamp` — stamp text via markpdf
- `markpdf image overlay` — overlay image via markpdf

## Usage Examples
- "markpdf --help"
- "markpdf <args>"

## Installation

```bash
go install github.com/ajaxray/markpdf@latest
```

## Examples

```bash
markpdf --version
markpdf --help
```

## Key Features
- pdf\n- watermark\n- documents
