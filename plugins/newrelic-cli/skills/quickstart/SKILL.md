# Newrelic-cli Plugin

Command line interface for New Relic observability platform.

## Commands

### Operations
- `newrelic-cli apm list` — list apm via newrelic
- `newrelic-cli entity search` — search entity via newrelic
- `newrelic-cli nerdgraph query` — query nerdgraph via newrelic

## Usage Examples
- "newrelic-cli --help"
- "newrelic-cli <args>"

## Installation

```bash
brew install newrelic-cli
```

## Examples

```bash
newrelic --version
newrelic --help
```
