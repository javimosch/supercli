# Intelli-shell Plugin

Like IntelliSense but for shells with command suggestions.

## Commands

### Operations
- `intelli-shell command suggest` — suggest command via intelli-shell
- `intelli-shell history search` — search history via intelli-shell
- `intelli-shell alias create` — create alias via intelli-shell
- `intelli-shell alias list` — list alias via intelli-shell
- `intelli-shell completion generate` — generate completion via intelli-shell
- `intelli-shell config init` — init config via intelli-shell
- `intelli-shell config show` — show config via intelli-shell

## Usage Examples
- "intelli-shell --help"
- "intelli-shell <args>"

## Installation

```bash
cargo install intelli-shell
```

## Examples

```bash
intelli-shell --version
intelli-shell --help
```
