# Speedtest-go Plugin

Go based Speedtest CLI.

## Commands

### Operations
- `speedtest-go network measure` — measure network via speedtest-go

## Usage Examples
- "speedtest-go --help"
- "speedtest-go <args>"

## Installation

```bash
brew install speedtest-go
```

## Examples

```bash
speedtest-go --version
speedtest-go --help
```
