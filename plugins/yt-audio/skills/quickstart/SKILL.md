# Yt-audio Plugin

Simple configurable youtube-dl wrapper to download and manage youtube audio.

## Commands

### Operations
- `yt-audio audio download` — download audio via yt-audio
- `yt-audio audio list` — list audio via yt-audio
- `yt-audio audio remove` — remove audio via yt-audio
- `yt-audio format set` — set format via yt-audio
- `yt-audio quality set` — set quality via yt-audio
- `yt-audio output set` — set output via yt-audio
- `yt-audio verbose set` — set verbose via yt-audio
- `yt-audio help show` — show help via yt-audio

## Usage Examples
- "yt-audio --help"
- "yt-audio <args>"

## Installation

```bash
pip install yt-audio
```
