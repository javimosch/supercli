# Nexe Plugin

Create a single executable out of your node.js apps.

## Commands

### Operations
- `nexe build compile` — compile build via nexe

## Usage Examples
- "nexe --help"
- "nexe <args>"

## Installation

```bash
npm install -g nexe
```

## Examples

```bash
nexe --version
nexe --help
```

## Key Features
- node\n- executable\n- compiler
