# Gomainr Plugin

CLI app that checks domain availability for keyword configurations.

## Commands

### Operations
- `gomainr domain check` — check domain via gomainr
- `gomainr keyword generate` — generate keyword via gomainr
- `gomainr config show` — show config via gomainr

## Usage Examples
- "gomainr --help"
- "gomainr <args>"

## Installation

```bash
go install github.com/MichaelThessel/gomainr@latest
```

## Examples

```bash
gomainr --version
gomainr --help
```

## Key Features
- domain\n- availability\n- networking
