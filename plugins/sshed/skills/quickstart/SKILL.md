# Sshed Plugin

SSH config editor and bookmarks manager.

## Commands

### Operations
- `sshed host add` — add host via sshed
- `sshed host list` — list host via sshed
- `sshed host connect` — connect host via sshed

## Usage Examples
- "sshed --help"
- "sshed <args>"

## Installation

```bash
go install github.com/trntv/sshed@latest
```

## Examples

```bash
sshed --version
sshed --help
```

## Key Features
- ssh\n- config\n- network
