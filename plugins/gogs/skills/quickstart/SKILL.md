# Gogs Plugin

Painless self-hosted Git service with CLI administration.

## Commands

### Operations
- `gogs admin create-user` — create-user admin via gogs
- `gogs admin delete-user` — delete-user admin via gogs
- `gogs web start` — start web via gogs

## Usage Examples
- "gogs --help"
- "gogs <args>"

## Installation

```bash
brew install gogs
```

## Examples

```bash
gogs --version
gogs --help
```

## Key Features
- git\n- self-hosted\n- scm
