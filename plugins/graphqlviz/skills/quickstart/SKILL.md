# Graphqlviz Plugin

Visualize GraphQL schemas as Graphviz graphs.

## Commands

### Operations
- `graphqlviz schema visualize` — visualize schema via graphqlviz

## Usage Examples
- "graphqlviz --help"
- "graphqlviz <args>"

## Installation

```bash
npm install -g graphqlviz
```

## Examples

```bash
graphqlviz --version
graphqlviz --help
```

## Key Features
- graphql\n- visualization\n- graphviz
