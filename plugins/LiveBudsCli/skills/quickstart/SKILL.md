# LiveBudsCli Plugin

Tool to control your Galaxy Buds+, Live, Buds Pro and Buds 2 Pro.

## Commands

### Operations
- `LiveBudsCli device connect` — connect device via LiveBudsCli
- `LiveBudsCli device disconnect` — disconnect device via LiveBudsCli
- `LiveBudsCli device list` — list device via LiveBudsCli
- `LiveBudsCli settings show` — show settings via LiveBudsCli
- `LiveBudsCli equalizer set` — set equalizer via LiveBudsCli
- `LiveBudsCli touchpad set` — set touchpad via LiveBudsCli
- `LiveBudsCli noise-canceling set` — set noise-canceling via LiveBudsCli
- `LiveBudsCli ambient-sound set` — set ambient-sound via LiveBudsCli
- `LiveBudsCli find-my-buds run` — run find-my-buds via LiveBudsCli
- `LiveBudsCli firmware info` — info firmware via LiveBudsCli
- `LiveBudsCli firmware update` — update firmware via LiveBudsCli

## Usage Examples

- "LiveBudsCli <args>"
