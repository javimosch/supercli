# Mcporter Plugin

Call MCPs via TypeScript masquerading as simple TypeScript API or package them as CLI.

## Commands

### Operations
- `mcporter mcp call` — call mcp via mcporter
- `mcporter mcp list` — list mcp via mcporter
- `mcporter api generate` — generate api via mcporter
- `mcporter cli package` — package cli via mcporter
- `mcporter config show` — show config via mcporter
- `mcporter verbose set` — set verbose via mcporter

## Usage Examples
- "mcporter --help"
- "mcporter <args>"

## Installation

```bash
npm install -g mcporter
```

## Examples

```bash
mcporter --version
mcporter --help
```

## Key Features
- mcp\n- typescript\n- api
