# Swagger-cli Plugin

Command line tools for Swagger/OpenAPI operations.

## Commands

### Operations
- `swagger-cli spec validate` — validate spec via swagger
- `swagger-cli spec bundle` — bundle spec via swagger

## Usage Examples
- "swagger-cli --help"
- "swagger-cli <args>"

## Installation

```bash
npm install -g swagger-cli
```

## Examples

```bash
swagger --version
swagger --help
```

## Key Features
- openapi\n- swagger\n- api
