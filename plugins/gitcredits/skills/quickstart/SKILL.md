# Gitcredits Plugin

Generate contributor credits and statistics for Git repositories.

## Commands

### Operations
- `gitcredits repo credits` — credits repo via gitcredits
- `gitcredits repo stats` — stats repo via gitcredits
- `gitcredits format export` — export format via gitcredits

## Usage Examples
- "gitcredits --help"
- "gitcredits <args>"

## Installation

```bash
go install github.com/Higangssh/gitcredits@latest
```

## Examples

```bash
gitcredits --version
gitcredits --help
```

## Key Features
- git\n- contributors\n- statistics
