# Npx_card Plugin

Display your npm card in terminal.

## Commands

### Operations
- `npx_card card display` — display card via npx-card

## Usage Examples
- "npx_card --help"
- "npx_card <args>"

## Installation

```bash
npx anmol098
```

## Examples

```bash
npx-card --version
npx-card --help
```

- npm\n- card\n- business-card
