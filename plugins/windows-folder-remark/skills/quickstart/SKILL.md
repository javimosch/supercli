# Windows-folder-remark Plugin

Lightweight CLI tool to add remarks to Windows folders via Desktop.ini privacy-first.

## Commands

### Operations
- `windows-folder-remark folder add` — add folder via folder-remark
- `windows-folder-remark folder remove` — remove folder via folder-remark
- `windows-folder-remark folder list` — list folder via folder-remark
- `windows-folder-remark remark show` — show remark via folder-remark
- `windows-folder-remark verbose set` — set verbose via folder-remark
- `windows-folder-remark help show` — show help via folder-remark

## Usage Examples
- "windows-folder-remark --help"
- "windows-folder-remark <args>"

## Installation

```bash
pip install windows-folder-remark
```
