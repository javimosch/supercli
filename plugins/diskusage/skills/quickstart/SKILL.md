# Diskusage Plugin

Show disk usage in terminal with visual charts.

## Commands

### Operations
- `diskusage path analyze` — analyze path via diskusage
- `diskusage tree show` — show tree via diskusage

## Usage Examples
- "diskusage --help"
- "diskusage <args>"

## Installation

```bash
go install github.com/chenquan/diskusage@latest
```

## Examples

```bash
diskusage --version
diskusage --help
```

## Key Features
- disk\n- filesystem\n- visualization
