# Site-audit-seo Plugin

SEO site audit tool for command line.

## Commands

### Operations
- `site-audit-seo site audit` — audit site via site-audit-seo

## Usage Examples
- "site-audit-seo --help"
- "site-audit-seo <args>"

## Installation

```bash
npm install -g site-audit-seo
```

## Examples

```bash
site-audit-seo --version
site-audit-seo --help
```

## Key Features
- seo\n- audit\n- web
