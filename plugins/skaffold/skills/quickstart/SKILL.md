# Skaffold Plugin

Easy and Repeatable Kubernetes Development.

## Commands

### Operations
- `skaffold dev run` — run dev via skaffold
- `skaffold build run` — run build via skaffold

## Usage Examples
- "skaffold --help"
- "skaffold <args>"

## Installation

```bash
brew install skaffold
```

## Examples

```bash
skaffold --version
skaffold --help
```

- kubernetes\n- k8s\n- devops
