# Inlets Plugin

Expose your local endpoints to the Internet.

## Commands

### Operations
- `inlets tunnel create` — create tunnel via inlets
- `inlets tunnel delete` — delete tunnel via inlets

## Usage Examples
- "inlets --help"
- "inlets <args>"

## Installation

```bash
brew install inlets
```

## Examples

```bash
inlets --version
inlets --help
```

## Key Features
- tunnel\n- reverse-proxy\n- networking
