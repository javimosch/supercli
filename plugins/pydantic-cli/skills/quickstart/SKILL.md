# Pydantic-cli Plugin

Turn Pydantic defined Data Models into CLI Tools with automatic argument parsing.

## Commands

### Operations
- `pydantic-cli model generate` — generate model via pydantic-cli
- `pydantic-cli model validate` — validate model via pydantic-cli
- `pydantic-cli schema show` — show schema via pydantic-cli
- `pydantic-cli config set` — set config via pydantic-cli
- `pydantic-cli output set` — set output via pydantic-cli
- `pydantic-cli verbose set` — set verbose via pydantic-cli
- `pydantic-cli help show` — show help via pydantic-cli

## Usage Examples
- "pydantic-cli --help"
- "pydantic-cli <args>"

## Installation

```bash
pip install pydantic-cli
```

## Examples

```bash
pydantic-cli --version
pydantic-cli --help
```
