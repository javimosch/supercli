# Go-passbolt-cli Plugin

Command line client for Passbolt password manager.

## Commands

### Operations
- `go-passbolt-cli resource list` — list resource via passbolt
- `go-passbolt-cli resource get` — get resource via passbolt
- `go-passbolt-cli folder list` — list folder via passbolt
- `go-passbolt-cli user list` — list user via passbolt

## Usage Examples
- "go-passbolt-cli --help"
- "go-passbolt-cli <args>"

## Installation

```bash
go install github.com/passbolt/go-passbolt-cli@latest
```

## Examples

```bash
passbolt --version
passbolt --help
```
