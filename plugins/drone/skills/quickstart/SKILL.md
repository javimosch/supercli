# Drone Plugin

Continuous integration and delivery platform CLI.

## Commands

### Operations
- `drone repo sync` — sync repo via drone
- `drone build list` — list build via drone

## Usage Examples
- "drone --help"
- "drone <args>"

## Installation

```bash
brew install drone
```

## Examples

```bash
drone --version
drone --help
```

## Key Features
- ci-cd\n- devops\n- automation
