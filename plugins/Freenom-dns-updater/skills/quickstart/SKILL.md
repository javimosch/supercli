# Freenom-dns-updater Plugin

Tool to update Freenoms DNS records automatically.

## Commands

### Operations
- `Freenom-dns-updater dns update` — update dns via freenom-dns-updater
- `Freenom-dns-updater domain list` — list domain via freenom-dns-updater
- `Freenom-dns-updater record add` — add record via freenom-dns-updater
- `Freenom-dns-updater record remove` — remove record via freenom-dns-updater
- `Freenom-dns-updater config show` — show config via freenom-dns-updater
- `Freenom-dns-updater verbose set` — set verbose via freenom-dns-updater
- `Freenom-dns-updater help show` — show help via freenom-dns-updater

## Usage Examples
- "Freenom-dns-updater --help"
- "Freenom-dns-updater <args>"
