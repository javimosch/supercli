# Kubectx Plugin

Faster way to switch between clusters and namespaces in kubectl.

## Commands

### Operations
- `kubectx context switch` — switch context via kubectx

## Usage Examples
- "kubectx --help"
- "kubectx <args>"

## Installation

```bash
brew install kubectx
```

## Examples

```bash
kubectx --version
kubectx --help
```

## Key Features
- kubernetes\n- k8s\n- context
