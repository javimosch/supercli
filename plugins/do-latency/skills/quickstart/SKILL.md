# Do-latency Plugin

Digital Ocean regions latency checker to find the fastest region for your needs.

## Commands

### Operations
- `do-latency latency check` — check latency via do-latency
- `do-latency region list` — list region via do-latency
- `do-latency region test` — test region via do-latency
- `do-latency output set` — set output via do-latency
- `do-latency verbose set` — set verbose via do-latency
- `do-latency help show` — show help via do-latency

## Usage Examples
- "do-latency --help"
- "do-latency <args>"

## Installation

```bash
pip install do-latency
```
