# Tldr Plugin

Simplified and community-driven man pages.

## Commands

### Operations
- `tldr page show` — show page via tldr

## Usage Examples
- "tldr --help"
- "tldr <args>"

## Installation

```bash
npm install -g tldr
```

## Examples

```bash
tldr --version
tldr --help
```

## Key Features
- documentation\n- help\n- man-pages
