# Trash-cli Plugin

Move files and folders to the trash.

## Commands

### Operations
- `trash-cli file trash` — trash file via trash

## Usage Examples
- "trash-cli --help"
- "trash-cli <args>"

## Installation

```bash
npm install -g trash-cli
```

## Examples

```bash
trash --version
trash --help
```

## Key Features
- filesystem\n- trash\n- cleanup
