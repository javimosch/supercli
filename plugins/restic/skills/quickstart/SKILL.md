# Restic Plugin

Fast, secure, efficient backup program.

## Commands

### Operations
- `restic snapshot list` — list snapshot via restic
- `restic backup run` — run backup via restic

## Usage Examples
- "restic --help"
- "restic <args>"

## Installation

```bash
brew install restic
```

## Examples

```bash
restic --version
restic --help
```

## Key Features
- backup\n- archive\n- security
