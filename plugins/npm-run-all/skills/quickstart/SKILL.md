# Npm-run-all Plugin

Run multiple npm-scripts in parallel or sequential.

## Commands

### Operations
- `npm-run-all script run` — run script via run-p

## Usage Examples
- "npm-run-all --help"
- "npm-run-all <args>"

## Installation

```bash
npm install -g npm-run-all
```

## Examples

```bash
run-p --version
run-p --help
```

## Key Features
- npm\n- scripts\n- parallel
