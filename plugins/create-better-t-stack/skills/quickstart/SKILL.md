# Create-better-t-stack Plugin

Modern CLI tool for scaffolding end-to-end type-safe TypeScript projects with best practices.

## Commands

### Operations
- `create-better-t-stack project create` — create project via create-better-t-stack
- `create-better-t-stack template list` — list template via create-better-t-stack
- `create-better-t-stack config set` — set config via create-better-t-stack
- `create-better-t-stack verbose set` — set verbose via create-better-t-stack
- `create-better-t-stack help show` — show help via create-better-t-stack

## Usage Examples
- "create-better-t-stack --help"
- "create-better-t-stack <args>"

## Installation

```bash
npm install -g create-better-t-stack
```

## Examples

```bash
create-better-t-stack --version
```
