# Chdb-go Plugin

ClickHouse CLI and client library.

## Commands

### Operations
- `chdb-go query execute` — execute query via chdb

## Usage Examples
- "chdb-go --help"
- "chdb-go <args>"

## Installation

```bash
go install github.com/chdb-io/chdb-go/cmd/chdb@latest
```

## Examples

```bash
chdb --version
chdb --help
```

## Key Features
- clickhouse\n- database\n- sql
