# Gh-stars Plugin

List and manage your GitHub starred repositories.

## Commands

### Operations
- `gh-stars stars list` — list stars via gh-stars
- `gh-stars stars sync` — sync stars via gh-stars
- `gh-stars stars search` — search stars via gh-stars

## Usage Examples
- "gh-stars --help"
- "gh-stars <args>"

## Installation

```bash
go install github.com/gkze/gh-stars@latest
```

## Examples

```bash
gh-stars --version
gh-stars --help
```

## Key Features
- github\n- stars\n- repositories
