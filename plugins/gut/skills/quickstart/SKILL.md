# Gut Plugin

A beginner friendly CLI for git.

## Commands

### Operations
- `gut repo init` — init repo via gut
- `gut commit create` — create commit via gut
- `gut branch switch` — switch branch via gut

## Usage Examples
- "gut --help"
- "gut <args>"

## Installation

```bash
brew install julien040/tap/gut
```

## Examples

```bash
gut --version
gut --help
```

## Key Features
- git\n- version-control\n- beginner-friendly
