# Compose-generator Plugin

Easy to use CLI tool to generate Docker Compose configurations.

## Commands

### Operations
- `compose-generator compose generate` — generate compose via compose-generator

## Usage Examples
- "compose-generator --help"
- "compose-generator <args>"

## Installation

```bash
brew install compose-generator
```

## Examples

```bash
compose-generator --version
compose-generator --help
```

## Key Features
- docker\n- docker-compose\n- generator
