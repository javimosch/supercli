# Rnx-kit Plugin

Modern scalable tools for React Native with exceptional developer experience.

## Commands

### Operations
- `rnx-kit app create` — create app via rnx
- `rnx-kit app configure` — configure app via rnx
- `rnx-kit doctor run` — run doctor via rnx
- `rnx-kit upgrade run` — run upgrade via rnx
- `rnx-kit config show` — show config via rnx
- `rnx-kit verbose set` — set verbose via rnx
- `rnx-kit help show` — show help via rnx

## Usage Examples
- "rnx-kit --help"
- "rnx-kit <args>"

## Installation

```bash
npm install -g @rnx-kit/cli
```
