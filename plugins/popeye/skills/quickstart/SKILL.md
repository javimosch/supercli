# Popeye Plugin

Kubernetes cluster resource sanitizer and linter.

## Commands

### Operations
- `popeye cluster scan` — scan cluster via popeye

## Usage Examples
- "popeye --help"
- "popeye <args>"

## Installation

```bash
brew install popeye
```

## Examples

```bash
popeye --version
popeye --help
```
