# Clasp Plugin

Command Line Apps Script Projects for Google Apps Script.

## Commands

### Operations
- `clasp project clone` — clone project via clasp
- `clasp project push` — push project via clasp
- `clasp project pull` — pull project via clasp
- `clasp project status` — status project via clasp
- `clasp script list` — list script via clasp
- `clasp script deploy` — deploy script via clasp
- `clasp script open` — open script via clasp
- `clasp logs watch` — watch logs via clasp
- `clasp config show` — show config via clasp
- `clasp verbose set` — set verbose via clasp

## Usage Examples
- "clasp --help"
- "clasp <args>"

## Installation

```bash
npm install -g @google/clasp
```

## Examples

```bash
clasp --version
clasp --help
```
