# QuickWall Plugin

Set latest wallpapers from Unsplash from the commandline.

## Commands

### Operations
- `QuickWall wallpaper set` — set wallpaper via quickwall
- `QuickWall wallpaper random` — random wallpaper via quickwall
- `QuickWall wallpaper search` — search wallpaper via quickwall
- `QuickWall category set` — set category via quickwall
- `QuickWall output set` — set output via quickwall
- `QuickWall verbose set` — set verbose via quickwall
- `QuickWall help show` — show help via quickwall

## Usage Examples
- "QuickWall --help"
- "QuickWall <args>"

## Installation

```bash
pip install quickwall
```

## Examples

```bash
quickwall --version
quickwall --help
```

- wallpaper\n- unsplash\n- desktop
