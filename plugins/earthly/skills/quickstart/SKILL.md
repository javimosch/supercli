# Earthly Plugin

Build automation tool that works with Dockerfiles.

## Commands

### Operations
- `earthly build run` — run build via earthly
- `earthly target list` — list target via earthly

## Usage Examples
- "earthly --help"
- "earthly <args>"

## Installation

```bash
brew install earthly
```

## Examples

```bash
earthly --version
earthly --help
```

## Key Features
- build\n- docker\n- ci-cd
