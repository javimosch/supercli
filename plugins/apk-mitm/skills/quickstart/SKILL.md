# Apk-mitm Plugin

Prepare APKs for HTTPS inspection.

## Commands

### Operations
- `apk-mitm apk patch` — patch apk via apk-mitm

## Usage Examples
- "apk-mitm --help"
- "apk-mitm <args>"

## Installation

```bash
npm install -g apk-mitm
```

## Examples

```bash
apk-mitm --version
apk-mitm --help
```

## Key Features
- android\n- apk\n- security
