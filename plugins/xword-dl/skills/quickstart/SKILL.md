# Xword-dl Plugin

Command line tool to scrape crosswords from online solvers and save them as puz files.

## Commands

### Operations
- `xword-dl crossword download` — download crossword via xword-dl
- `xword-dl crossword list` — list crossword via xword-dl
- `xword-dl source set` — set source via xword-dl
- `xword-dl output set` — set output via xword-dl
- `xword-dl format set` — set format via xword-dl
- `xword-dl verbose set` — set verbose via xword-dl
- `xword-dl help show` — show help via xword-dl

## Usage Examples
- "xword-dl --help"
- "xword-dl <args>"

## Installation

```bash
pip install xword-dl
```
