# Klepto Plugin

Tool for copying and anonymizing database data.

## Commands

### Operations
- `klepto database dump` — dump database via klepto
- `klepto database steal` — steal database via klepto
- `klepto config init` — init config via klepto
- `klepto config verify` — verify config via klepto

## Usage Examples
- "klepto --help"
- "klepto <args>"

## Installation

```bash
go install github.com/hellofresh/klepto@latest
```

## Examples

```bash
klepto --version
klepto --help
```
