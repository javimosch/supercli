# EasyCert Plugin

Simple tool for creating SSL/TLS certificates for development.

## Commands

### Operations
- `EasyCert cert create` — create cert via easycert
- `EasyCert cert verify` — verify cert via easycert
- `EasyCert cert renew` — renew cert via easycert

## Usage Examples
- "EasyCert --help"
- "EasyCert <args>"

## Installation

```bash
go install github.com/deckarep/EasyCert@latest
```

## Examples

```bash
easycert --version
easycert --help
```

## Key Features
- ssl\n- tls\n- development
