# Wikit Plugin

Command line Wikipedia summary tool.

## Commands

### Operations
- `wikit query search` — search query via wikit

## Usage Examples
- "wikit --help"
- "wikit <args>"

## Installation

```bash
npm install -g wikit
```

## Examples

```bash
wikit --version
wikit --help
```

## Key Features
- wikipedia\n- search\n- knowledge
