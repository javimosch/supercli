# Pluto Plugin

Finds deprecated Kubernetes API versions in your cluster.

## Commands

### Operations
- `pluto cluster detect` — detect cluster via pluto
- `pluto files detect` — detect files via pluto

## Usage Examples
- "pluto --help"
- "pluto <args>"

## Installation

```bash
brew install FairwindsOps/tap/pluto
```

## Examples

```bash
pluto --version
pluto --help
```

## Key Features
- kubernetes\n- deprecation\n- api-versions
