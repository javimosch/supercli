# Pagit Plugin

Powerful CLI tool for managing git repositories as pages.

## Commands

### Operations
- `pagit repo add` — add repo via pagit
- `pagit repo remove` — remove repo via pagit
- `pagit repo list` — list repo via pagit
- `pagit repo sync` — sync repo via pagit
- `pagit page create` — create page via pagit
- `pagit page publish` — publish page via pagit
- `pagit config init` — init config via pagit
- `pagit config show` — show config via pagit
- `pagit status show` — show status via pagit
- `pagit log show` — show log via pagit

## Usage Examples
- "pagit --help"
- "pagit <args>"

## Installation

```bash
npm install -g pagit
```

## Examples

```bash
pagit --version
pagit --help
```
