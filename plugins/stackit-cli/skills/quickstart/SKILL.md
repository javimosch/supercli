# Stackit-cli Plugin

Command line interface to manage STACKIT cloud resources.

## Commands

### Operations
- `stackit-cli project list` — list project via stackit
- `stackit-cli resource list` — list resource via stackit
- `stackit-cli config set` — set config via stackit

## Usage Examples
- "stackit-cli --help"
- "stackit-cli <args>"

## Installation

```bash
brew install stackit
```

## Examples

```bash
stackit --version
stackit --help
```
