# Go-qrcode Plugin

QR code generator with terminal and image output.

## Commands

### Operations
- `go-qrcode data encode` — encode data via go-qrcode

## Usage Examples
- "go-qrcode --help"
- "go-qrcode <args>"

## Installation

```bash
go install github.com/yeqown/go-qrcode/cmd/go-qrcode@latest
```

## Examples

```bash
go-qrcode --version
go-qrcode --help
```

## Key Features
- qrcode\n- generator\n- images
