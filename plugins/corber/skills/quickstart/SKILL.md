# Corber Plugin

Cordova and Electron hybrid app development CLI.

## Commands

### Operations
- `corber app build` — build app via corber
- `corber app serve` — serve app via corber
- `corber app init` — init app via corber

## Usage Examples
- "corber --help"
- "corber <args>"

## Installation

```bash
npm install -g corber
```

## Examples

```bash
corber --version
corber --help
```

## Key Features
- cordova\n- electron\n- hybrid-app
