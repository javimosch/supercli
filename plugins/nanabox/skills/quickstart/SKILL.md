# Nanabox Plugin

Local development environment and deployment platform.

## Commands

### Operations
- `nanabox app run` — run app via nanobox
- `nanabox app deploy` — deploy app via nanobox

## Usage Examples
- "nanabox --help"
- "nanabox <args>"

## Installation

```bash
brew install nanobox
```

## Examples

```bash
nanobox --version
nanobox --help
```

- development\n- deployment\n- containers
