# Lux Plugin

Fast and simple video download library and CLI tool.

## Commands

### Operations
- `lux video download` — download video via lux

## Usage Examples
- "lux --help"
- "lux <args>"

## Installation

```bash
go install github.com/iawia002/lux@latest
```

## Examples

```bash
lux --version
lux --help
```
