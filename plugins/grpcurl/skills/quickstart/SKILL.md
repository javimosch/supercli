# Grpcurl Plugin

Command-line tool for interacting with gRPC servers.

## Commands

### Operations
- `grpcurl server list` — list server via grpcurl
- `grpcurl server describe` — describe server via grpcurl
- `grpcurl server invoke` — invoke server via grpcurl

## Usage Examples
- "grpcurl --help"
- "grpcurl <args>"

## Installation

```bash
brew install grpcurl
```

## Examples

```bash
grpcurl --version
grpcurl --help
```

## Key Features
- grpc\n- api\n- curl
