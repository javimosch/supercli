# Igniteui-cli Plugin

Ignite UI Command-Line Interface by Infragistics.

## Commands

### Operations
- `igniteui-cli project new` — new project via igniteui
- `igniteui-cli project generate` — generate project via igniteui
- `igniteui-cli component add` — add component via igniteui
- `igniteui-cli component list` — list component via igniteui
- `igniteui-cli template list` — list template via igniteui
- `igniteui-cli template use` — use template via igniteui
- `igniteui-cli config init` — init config via igniteui
- `igniteui-cli config show` — show config via igniteui
- `igniteui-cli build run` — run build via igniteui
- `igniteui-cli serve start` — start serve via igniteui
- `igniteui-cli upgrade run` — run upgrade via igniteui
- `igniteui-cli doc generate` — generate doc via igniteui
- `igniteui-cli lint run` — run lint via igniteui
- `igniteui-cli test run` — run test via igniteui

## Usage Examples
- "igniteui-cli --help"
- "igniteui-cli <args>"
