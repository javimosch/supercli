# Zfind Plugin

Search for files (even inside tar/zip/7z/rar) using a SQL-WHERE filter.

## Commands

### Operations
- `zfind file find` — find file via zfind

## Usage Examples
- "zfind --help"
- "zfind <args>"

## Installation

```bash
brew install laktak/tap/zfind
```

## Examples

```bash
zfind --version
zfind --help
```

## Key Features
- search\n- sql\n- archives
