# Sandy Plugin

Process tracing and syscall monitoring CLI.

## Commands

### Operations
- `sandy process trace` — trace process via sandy

## Usage Examples
- "sandy --help"
- "sandy <args>"

## Installation

```bash
go install github.com/craigmulligan/sandy@latest
```

## Examples

```bash
sandy --version
sandy --help
```

## Key Features
- ptrace\n- syscall\n- monitoring
