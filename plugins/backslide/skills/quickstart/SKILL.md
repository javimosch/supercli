# Backslide Plugin

CLI tool for creating and presenting Markdown-based slides.

## Commands

### Operations
- `backslide slides create` — create slides via backslide
- `backslide slides serve` — serve slides via backslide
- `backslide slides export` — export slides via backslide

## Usage Examples
- "backslide --help"
- "backslide <args>"

## Installation

```bash
npm install -g backslide
```

## Examples

```bash
backslide --version
backslide --help
```

## Key Features
- markdown\n- presentation\n- slides
