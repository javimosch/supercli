# Openhue-cli Plugin

Command line tool for Philips Hue smart lights.

## Commands

### Operations
- `openhue-cli light list` — list light via openhue
- `openhue-cli light set` — set light via openhue
- `openhue-cli room list` — list room via openhue
- `openhue-cli scene list` — list scene via openhue

## Usage Examples
- "openhue-cli --help"
- "openhue-cli <args>"

## Installation

```bash
brew install openhue-cli
```

## Examples

```bash
openhue --version
openhue --help
```
