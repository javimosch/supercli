# Editly Plugin

Slick, declarative command line video editing.

## Commands

### Operations
- `editly video edit` — edit video via editly

## Usage Examples
- "editly --help"
- "editly <args>"

## Installation

```bash
npm install -g editly
```

## Examples

```bash
editly --version
editly --help
```

## Key Features
- video\n- ffmpeg\n- editing
