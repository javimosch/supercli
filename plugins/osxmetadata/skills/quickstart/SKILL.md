# Osxmetadata Plugin

Python package to read and write MacOS extended attribute metadata such as tags and Finder comments.

## Commands

### Operations
- `osxmetadata file read` — read file via osxmetadata
- `osxmetadata file write` — write file via osxmetadata
- `osxmetadata tag add` — add tag via osxmetadata
- `osxmetadata tag remove` — remove tag via osxmetadata
- `osxmetadata comment set` — set comment via osxmetadata
- `osxmetadata verbose set` — set verbose via osxmetadata
- `osxmetadata help show` — show help via osxmetadata

## Usage Examples
- "osxmetadata --help"
- "osxmetadata <args>"

## Installation

```bash
pip install osxmetadata
```

## Examples

```bash
osxmetadata --version
osxmetadata --help
```
