# Travis-watch Plugin

Watch Travis CI build status from the terminal.

## Commands

### Operations
- `travis-watch build watch` — watch build via travis-watch

## Usage Examples
- "travis-watch --help"
- "travis-watch <args>"

## Installation

```bash
npm install -g travis-watch
```

## Examples

```bash
travis-watch --version
travis-watch --help
```

## Key Features
- travis-ci\n- ci\n- monitoring
