# Grmon Plugin

Command line monitoring for goroutines.

## Commands

### Operations
- `grmon process monitor` — monitor process via grmon

## Usage Examples
- "grmon --help"
- "grmon <args>"

## Installation

```bash
go install github.com/bcicen/grmon@latest
```

## Examples

```bash
grmon --version
grmon --help
```
