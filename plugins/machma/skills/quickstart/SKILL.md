# Machma Plugin

Parallel command execution tool with progress bars.

## Commands

### Operations
- `machma command run` — run command via machma

## Usage Examples
- "machma --help"
- "machma <args>"

## Installation

```bash
go install github.com/fd0/machma@latest
```

## Examples

```bash
machma --version
machma --help
```

## Key Features
- parallel\n- execution\n- progress
