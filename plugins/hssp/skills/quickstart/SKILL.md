# Hssp Plugin

HTTP server security probe and header analyzer.

## Commands

### Operations
- `hssp url probe` — probe url via hssp
- `hssp url scan` — scan url via hssp

## Usage Examples
- "hssp --help"
- "hssp <args>"

## Installation

```bash
go install github.com/sterchelen/hssp@latest
```

## Examples

```bash
hssp --version
hssp --help
```

## Key Features
- security\n- http\n- headers
