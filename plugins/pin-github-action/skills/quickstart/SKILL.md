# Pin-github-action Plugin

Pin GitHub Actions to specific commit SHAs.

## Commands

### Operations
- `pin-github-action file pin` — pin file via pin-github-action

## Usage Examples
- "pin-github-action --help"
- "pin-github-action <args>"

## Installation

```bash
npm install -g pin-github-action
```

## Examples

```bash
pin-github-action --version
pin-github-action --help
```

- github-actions\n- security\n- ci
