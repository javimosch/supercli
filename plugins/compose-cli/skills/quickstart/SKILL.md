# Compose-cli Plugin

Docker Compose CLI for multi-container applications.

## Commands

### Operations
- `compose-cli stack up` — up stack via docker-compose
- `compose-cli stack down` — down stack via docker-compose

## Usage Examples
- "compose-cli --help"
- "compose-cli <args>"

## Installation

```bash
brew install docker-compose
```

## Examples

```bash
docker-compose --version
docker-compose --help
```

## Key Features
- docker\n- containers\n- orchestration
