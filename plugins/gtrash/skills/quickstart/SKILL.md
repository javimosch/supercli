# Gtrash Plugin

Trash CLI that moves files to system trash instead of deleting.

## Commands

### Operations
- `gtrash file trash` — trash file via gtrash
- `gtrash file restore` — restore file via gtrash
- `gtrash file list` — list file via gtrash
- `gtrash file empty` — empty file via gtrash

## Usage Examples
- "gtrash --help"
- "gtrash <args>"

## Installation

```bash
brew install gtrash
```

## Examples

```bash
gtrash --version
gtrash --help
```
