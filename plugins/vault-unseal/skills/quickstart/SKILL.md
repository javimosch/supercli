# Vault-unseal Plugin

Automated HashiCorp Vault unseal utility.

## Commands

### Operations
- `vault-unseal vault unseal` — unseal vault via vault-unseal
- `vault-unseal vault status` — status vault via vault-unseal

## Usage Examples
- "vault-unseal --help"
- "vault-unseal <args>"

## Installation

```bash
go install github.com/lrstanley/vault-unseal@latest
```

## Examples

```bash
vault-unseal --version
vault-unseal --help
```
