# Age Plugin

A simple, modern and secure encryption tool with small explicit keys.

## Commands

### Operations
- `age file encrypt` — encrypt file via age
- `age file decrypt` — decrypt file via age

## Usage Examples
- "age --help"
- "age <args>"

## Installation

```bash
brew install age
```

## Examples

```bash
age --version
age --help
```
