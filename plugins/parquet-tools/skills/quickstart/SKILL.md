# Parquet-tools Plugin

Easy install parquet-tools for working with Parquet files from command line.

## Commands

### Operations
- `parquet-tools file show` — show file via parquet-tools
- `parquet-tools file convert` — convert file via parquet-tools
- `parquet-tools schema show` — show schema via parquet-tools
- `parquet-tools metadata show` — show metadata via parquet-tools
- `parquet-tools output set` — set output via parquet-tools
- `parquet-tools verbose set` — set verbose via parquet-tools
- `parquet-tools help show` — show help via parquet-tools

## Usage Examples
- "parquet-tools --help"
- "parquet-tools <args>"

## Installation

```bash
pip install parquet-tools
```

## Examples

```bash
parquet-tools --version
parquet-tools --help
```
