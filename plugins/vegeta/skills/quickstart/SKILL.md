# Vegeta Plugin

HTTP load testing tool and library.

## Commands

### Operations
- `vegeta attack run` — run attack via vegeta
- `vegeta report generate` — generate report via vegeta

## Usage Examples
- "vegeta --help"
- "vegeta <args>"

## Installation

```bash
go install github.com/tsenart/vegeta@latest
```

## Examples

```bash
vegeta --version
vegeta --help
```

## Key Features
- http\n- benchmark\n- load-testing
