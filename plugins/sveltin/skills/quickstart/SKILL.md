# Sveltin Plugin

CLI tool for building SvelteKit static websites.

## Commands

### Operations
- `sveltin site create` — create site via sveltin
- `sveltin site build` — build site via sveltin
- `sveltin site deploy` — deploy site via sveltin
- `sveltin content add` — add content via sveltin
- `sveltin resource add` — add resource via sveltin

## Usage Examples
- "sveltin --help"
- "sveltin <args>"

## Installation

```bash
go install github.com/sveltinio/sveltin@latest
```

## Examples

```bash
sveltin --version
sveltin --help
```

## Key Features
- svelte\n- static-site\n- generator
