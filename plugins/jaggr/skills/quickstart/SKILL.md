# Jaggr Plugin

JSON aggregator and histogram viewer.

## Commands

### Operations
- `jaggr data aggregate` — aggregate data via jaggr

## Usage Examples
- "jaggr --help"
- "jaggr <args>"

## Installation

```bash
go install github.com/rs/jaggr@latest
```

## Examples

```bash
jaggr --version
jaggr --help
```

## Key Features
- json\n- aggregation\n- charts
