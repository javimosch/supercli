# Snm Plugin

Smol and simple Node.js version manager written in Rust.

## Commands

### Operations
- `snm node install` — install node via snm
- `snm node list` — list node via snm
- `snm node use` — use node via snm
- `snm node uninstall` — uninstall node via snm
- `snm node current` — current node via snm
- `snm node alias` — alias node via snm
- `snm node lts` — lts node via snm

## Usage Examples
- "snm --help"
- "snm <args>"

## Installation

```bash
cargo install snm
```

## Examples

```bash
snm --version
snm --help
```
