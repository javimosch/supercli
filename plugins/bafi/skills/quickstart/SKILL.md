# Bafi Plugin

Command line tool for formatting and beautifying text output.

## Commands

### Operations
- `bafi text format` — format text via bafi
- `bafi json beautify` — beautify json via bafi
- `bafi xml beautify` — beautify xml via bafi
- `bafi yaml beautify` — beautify yaml via bafi

## Usage Examples
- "bafi --help"
- "bafi <args>"

## Installation

```bash
go install github.com/mritd/bafi@latest
```

## Examples

```bash
bafi --version
bafi --help
```
