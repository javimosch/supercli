# Gomockhandler Plugin

Manage and generate Go mock implementations.

## Commands

### Operations
- `gomockhandler mock generate` — generate mock via gomockhandler
- `gomockhandler mock check` — check mock via gomockhandler
- `gomockhandler mock list` — list mock via gomockhandler

## Usage Examples
- "gomockhandler --help"
- "gomockhandler <args>"

## Installation

```bash
go install github.com/sanposhiho/gomockhandler@latest
```

## Examples

```bash
gomockhandler --version
gomockhandler --help
```

- go\n- mock\n- testing
