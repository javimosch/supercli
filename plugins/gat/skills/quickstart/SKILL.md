# Gat Plugin

Pretty-print files in terminal with syntax highlighting.

## Commands

### Operations
- `gat file print` — print file via gat

## Usage Examples
- "gat --help"
- "gat <args>"

## Installation

```bash
brew install gat
```

## Examples

```bash
gat --version
gat --help
```

## Key Features
- cat\n- syntax-highlighting\n- terminal
