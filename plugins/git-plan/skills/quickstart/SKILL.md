# Git-plan Plugin

Git Plan a better workflow for git with planning and tracking features.

## Commands

### Operations
- `git-plan plan create` — create plan via git-plan
- `git-plan plan list` — list plan via git-plan
- `git-plan plan show` — show plan via git-plan
- `git-plan task add` — add task via git-plan
- `git-plan task complete` — complete task via git-plan
- `git-plan config show` — show config via git-plan
- `git-plan verbose set` — set verbose via git-plan
- `git-plan help show` — show help via git-plan

## Usage Examples
- "git-plan --help"
- "git-plan <args>"

## Installation

```bash
pip install git-plan
```
