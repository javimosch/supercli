# Undock Plugin

Extract contents of a container image in a local folder.

## Commands

### Operations
- `undock image extract` — extract image via undock

## Usage Examples
- "undock --help"
- "undock <args>"

## Installation

```bash
brew install undock
```

## Examples

```bash
undock --version
undock --help
```

## Key Features
- containers\n- docker\n- extraction
