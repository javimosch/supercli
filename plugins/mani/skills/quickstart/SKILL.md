# Mani Plugin

CLI tool to help you manage multiple repositories.

## Commands

### Operations
- `mani project list` — list project via mani
- `mani run exec` — exec run via mani

## Usage Examples
- "mani --help"
- "mani <args>"

## Installation

```bash
brew install mani
```

## Examples

```bash
mani --version
mani --help
```

## Key Features
- git\n- repositories\n- monorepo
