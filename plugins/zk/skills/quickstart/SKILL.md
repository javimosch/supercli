# Zk Plugin

Plain text note-taking assistant.

## Commands

### Operations
- `zk note create` — create note via zk
- `zk note list` — list note via zk
- `zk note search` — search note via zk

## Usage Examples
- "zk --help"
- "zk <args>"

## Installation

```bash
brew install zk
```

## Examples

```bash
zk --version
zk --help
```

## Key Features
- notes\n- markdown\n- zettelkasten
