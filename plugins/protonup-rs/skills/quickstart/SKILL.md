# Protonup-rs Plugin

Automate installation and update of Linux Gaming Compatibility tools.

## Commands

### Operations
- `protonup-rs proton install` — install proton via protonup-rs
- `protonup-rs proton update` — update proton via protonup-rs
- `protonup-rs proton list` — list proton via protonup-rs
- `protonup-rs proton remove` — remove proton via protonup-rs
- `protonup-rs proton download` — download proton via protonup-rs
- `protonup-rs config show` — show config via protonup-rs

## Usage Examples
- "protonup-rs --help"
- "protonup-rs <args>"

## Installation

```bash
cargo install protonup-rs
```

## Examples

```bash
protonup-rs --version
protonup-rs --help
```
