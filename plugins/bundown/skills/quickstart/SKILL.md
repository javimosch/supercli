# Bundown Plugin

Fast all-in-one Markdown runtime and bundler built on Bun.

## Commands

### Operations
- `bundown markdown run` — run markdown via bundown
- `bundown file bundle` — bundle file via bundown
- `bundown file compile` — compile file via bundown
- `bundown file watch` — watch file via bundown
- `bundown config set` — set config via bundown
- `bundown output set` — set output via bundown
- `bundown config show` — show config via bundown
- `bundown verbose set` — set verbose via bundown

## Usage Examples
- "bundown --help"
- "bundown <args>"

## Installation

```bash
npm install -g bundown
```

## Examples

```bash
bundown --version
bundown --help
```
