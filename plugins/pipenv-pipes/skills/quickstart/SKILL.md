# Pipenv-pipes Plugin

PipEnv Environment Switcher for managing multiple Python virtual environments.

## Commands

### Operations
- `pipenv-pipes env list` — list env via pipenv-pipes
- `pipenv-pipes env activate` — activate env via pipenv-pipes
- `pipenv-pipes env create` — create env via pipenv-pipes
- `pipenv-pipes env delete` — delete env via pipenv-pipes
- `pipenv-pipes env switch` — switch env via pipenv-pipes
- `pipenv-pipes config show` — show config via pipenv-pipes
- `pipenv-pipes verbose set` — set verbose via pipenv-pipes
- `pipenv-pipes help show` — show help via pipenv-pipes

## Usage Examples
- "pipenv-pipes --help"
- "pipenv-pipes <args>"

## Installation

```bash
pip install pipenv-pipes
```
