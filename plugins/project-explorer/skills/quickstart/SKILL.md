# Project-explorer Plugin

Generate project directory tree and documentation.

## Commands

### Operations
- `project-explorer project explore` — explore project via project-explorer

## Usage Examples
- "project-explorer --help"
- "project-explorer <args>"

## Installation

```bash
npm install -g @sdras/project-explorer
```

## Examples

```bash
project-explorer --version
project-explorer --help
```

## Key Features
- documentation\n- tree\n- project
