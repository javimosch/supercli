# Periodic-table-cli Plugin

Periodic table of elements in your terminal.

## Commands

### Operations
- `periodic-table-cli element lookup` — lookup element via periodic-table

## Usage Examples
- "periodic-table-cli --help"
- "periodic-table-cli <args>"

## Installation

```bash
npm install -g periodic-table-cli
```

## Examples

```bash
periodic-table --version
periodic-table --help
```
