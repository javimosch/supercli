# Hygen Plugin

Fast, scalable code generator that lives in your project.

## Commands

### Operations
- `hygen generator run` — run generator via hygen

## Usage Examples
- "hygen --help"
- "hygen <args>"

## Installation

```bash
npm install -g hygen
```

## Examples

```bash
hygen --version
hygen --help
```

## Key Features
- code-generation\n- scaffold\n- templates
