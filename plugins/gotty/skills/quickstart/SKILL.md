# Gotty Plugin

Share your terminal as a web application.

## Commands

### Operations
- `gotty command run` — run command via gotty

## Usage Examples
- "gotty --help"
- "gotty <args>"

## Installation

```bash
go install github.com/yudai/gotty@latest
```

## Examples

```bash
gotty --version
gotty --help
```

## Key Features
- terminal\n- sharing\n- web
