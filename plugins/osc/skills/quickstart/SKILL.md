# Osc Plugin

Copy and paste text over SSH using OSC52 escape sequences.

## Commands

### Operations
- `osc clipboard copy` — copy clipboard via osc

## Usage Examples
- "osc --help"
- "osc <args>"

## Installation

```bash
brew install osc
```

## Examples

```bash
osc --version
osc --help
```

## Key Features
- clipboard\n- ssh\n- terminal
