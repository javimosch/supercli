# Extension-cli Plugin

Build and manage browser extensions.

## Commands

### Operations
- `extension-cli extension build` — build extension via extension

## Usage Examples
- "extension-cli --help"
- "extension-cli <args>"

## Installation

```bash
npm install -g extension-cli
```

## Examples

```bash
extension --version
extension --help
```

- browser-extension\n- chrome\n- firefox
