# Protolint Plugin

A pluggable linter and fixer to enforce Protocol Buffer style and conventions.

## Commands

### Operations
- `protolint file lint` — lint file via protolint
- `protolint file fix` — fix file via protolint

## Usage Examples
- "protolint --help"
- "protolint <args>"

## Installation

```bash
brew install protolint
```

## Examples

```bash
protolint --version
protolint --help
```
