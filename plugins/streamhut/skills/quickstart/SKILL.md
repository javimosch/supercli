# Streamhut Plugin

Real-time stream sharing over WebSockets.

## Commands

### Operations
- `streamhut stream share` — share stream via streamhut

## Usage Examples
- "streamhut --help"
- "streamhut <args>"

## Installation

```bash
npm install -g streamhut
```

## Examples

```bash
streamhut --version
streamhut --help
```

## Key Features
- streams\n- websockets\n- sharing
