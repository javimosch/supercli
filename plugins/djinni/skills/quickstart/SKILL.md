# Djinni Plugin

Generate cross-language type declarations and interface bindings for multiple languages.

## Commands

### Operations
- `djinni idl generate` — generate idl via djinni
- `djinni idl parse` — parse idl via djinni
- `djinni output set` — set output via djinni
- `djinni language set` — set language via djinni
- `djinni cpp generate` — generate cpp via djinni
- `djinni java generate` — generate java via djinni
- `djinni objc generate` — generate objc via djinni
- `djinni swift generate` — generate swift via djinni
- `djinni python generate` — generate python via djinni
- `djinni wasm generate` — generate wasm via djinni
- `djinni config show` — show config via djinni
- `djinni idl check` — check idl via djinni
- `djinni idl list` — list idl via djinni
- `djinni config show` — show config via djinni
- `djinni verbose set` — set verbose via djinni

## Usage Examples

- "djinni <args>"
