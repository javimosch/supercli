# Gitcs Plugin

Git commit statistics and contributor analysis CLI.

## Commands

### Operations
- `gitcs repo analyze` — analyze repo via gitcs
- `gitcs contributor stats` — stats contributor via gitcs
- `gitcs commit timeline` — timeline commit via gitcs

## Usage Examples
- "gitcs --help"
- "gitcs <args>"

## Installation

```bash
go install github.com/hrtsegv/gitcs@latest
```

## Examples

```bash
gitcs --version
gitcs --help
```

## Key Features
- git\n- statistics\n- contributors
