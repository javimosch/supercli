# Vininfo Plugin

Extracts useful information from Vehicle Identification Number VIN.

## Commands

### Operations
- `vininfo vin decode` — decode vin via vininfo
- `vininfo vin validate` — validate vin via vininfo
- `vininfo info show` — show info via vininfo
- `vininfo make show` — show make via vininfo
- `vininfo model show` — show model via vininfo
- `vininfo year show` — show year via vininfo
- `vininfo verbose set` — set verbose via vininfo
- `vininfo help show` — show help via vininfo

## Usage Examples
- "vininfo --help"
- "vininfo <args>"

## Installation

```bash
pip install vininfo
```

## Examples

```bash
vininfo --version
vininfo --help
```
