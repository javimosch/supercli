# Mongotail Plugin

Command line tool to log all MongoDB queries in a tailable format.

## Commands

### Operations
- `mongotail query tail` — tail query via mongotail
- `mongotail query filter` — filter query via mongotail
- `mongotail connection set` — set connection via mongotail
- `mongotail database set` — set database via mongotail
- `mongotail output set` — set output via mongotail
- `mongotail verbose set` — set verbose via mongotail
- `mongotail help show` — show help via mongotail

## Usage Examples
- "mongotail --help"
- "mongotail <args>"

## Installation

```bash
pip install mongotail
```

## Examples

```bash
mongotail --version
mongotail --help
```
