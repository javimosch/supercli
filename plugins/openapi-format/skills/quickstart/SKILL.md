# Openapi-format Plugin

Format and filter OpenAPI specification files.

## Commands

### Operations
- `openapi-format file format` — format file via openapi-format
- `openapi-format file filter` — filter file via openapi-format

## Usage Examples
- "openapi-format --help"
- "openapi-format <args>"

## Installation

```bash
npm install -g openapi-format
```

## Examples

```bash
openapi-format --version
openapi-format --help
```

## Key Features
- openapi\n- format\n- filter
