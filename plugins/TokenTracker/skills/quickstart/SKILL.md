# TokenTracker Plugin

Track and display cryptocurrency token prices.

## Commands

### Operations
- `TokenTracker token track` — track token via tokentracker
- `TokenTracker market list` — list market via tokentracker

## Usage Examples
- "TokenTracker --help"
- "TokenTracker <args>"

## Installation

```bash
npm install -g tokentracker
```

## Examples

```bash
tokentracker --version
tokentracker --help
```

## Key Features
- crypto\n- tokens\n- price-tracking
