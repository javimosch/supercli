# Noobaa-operator Plugin

Operator CLI for managing NooBaa object storage.

## Commands

### Operations
- `noobaa-operator system install` — install system via noobaa
- `noobaa-operator bucket list` — list bucket via noobaa
- `noobaa-operator backingstore list` — list backingstore via noobaa

## Usage Examples
- "noobaa-operator --help"
- "noobaa-operator <args>"

## Installation

```bash
brew install noobaa/noobaa-operator/noobaa-cli
```

## Examples

```bash
noobaa --version
noobaa --help
```
