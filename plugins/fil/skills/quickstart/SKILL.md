# Fil Plugin

Fast and simple file search tool.

## Commands

### Operations
- `fil file find` — find file via fil
- `fil directory list` — list directory via fil

## Usage Examples
- "fil --help"
- "fil <args>"

## Installation

```bash
go install github.com/file-go/fil@latest
```

## Examples

```bash
fil --version
fil --help
```

## Key Features
- file-search\n- find\n- filesystem
