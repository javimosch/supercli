# Cosign Plugin

Container Signing, Verification and Storage in an OCI registry.

## Commands

### Operations
- `cosign image sign` — sign image via cosign
- `cosign image verify` — verify image via cosign
- `cosign image attest` — attest image via cosign

## Usage Examples
- "cosign --help"
- "cosign <args>"

## Installation

```bash
brew install cosign
```

## Examples

```bash
cosign --version
cosign --help
```
