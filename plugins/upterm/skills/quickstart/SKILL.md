# Upterm Plugin

Secure terminal sharing with end-to-end encryption.

## Commands

### Operations
- `upterm host start` — start host via upterm
- `upterm client join` — join client via upterm

## Usage Examples
- "upterm --help"
- "upterm <args>"

## Installation

```bash
brew install owenthereal/upterm/upterm
```

## Examples

```bash
upterm --version
upterm --help
```

## Key Features
- terminal\n- ssh\n- sharing
