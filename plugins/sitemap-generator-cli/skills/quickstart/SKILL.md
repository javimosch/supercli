# Sitemap-generator-cli Plugin

Generate sitemap XML from a URL.

## Commands

### Operations
- `sitemap-generator-cli url generate` — generate url via sitemap-generator

## Usage Examples
- "sitemap-generator-cli --help"
- "sitemap-generator-cli <args>"

## Installation

```bash
npm install -g sitemap-generator-cli
```

## Examples

```bash
sitemap-generator --version
sitemap-generator --help
```

## Key Features
- sitemap\n- seo\n- xml
