# Zmk-viewer Plugin

Generate preview images from ZMK keymap files.

## Commands

### Operations
- `zmk-viewer keymap generate` — generate keymap via zmk-viewer
- `zmk-viewer board list` — list board via zmk-viewer

## Usage Examples
- "zmk-viewer --help"
- "zmk-viewer <args>"

## Installation

```bash
go install github.com/MrMarble/zmk-viewer@latest
```

## Examples

```bash
zmk-viewer --version
zmk-viewer --help
```

## Key Features
- zmk\n- keyboard\n- keymap
