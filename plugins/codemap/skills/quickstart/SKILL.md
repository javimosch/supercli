# Codemap Plugin

Generate code maps and visualization from source code.

## Commands

### Operations
- `codemap code map` — map code via codemap

## Usage Examples
- "codemap --help"
- "codemap <args>"

## Installation

```bash
go install github.com/JordanCoin/codemap@latest
```

## Examples

```bash
codemap --version
codemap --help
```

## Key Features
- code\n- visualization\n- map
