# Taskdog Plugin

Terminal task manager with intelligent schedule optimization keyboard-only.

## Commands

### Operations
- `taskdog task add` — add task via taskdog
- `taskdog task list` — list task via taskdog
- `taskdog task complete` — complete task via taskdog
- `taskdog task delete` — delete task via taskdog
- `taskdog schedule show` — show schedule via taskdog
- `taskdog config show` — show config via taskdog
- `taskdog verbose set` — set verbose via taskdog
- `taskdog help show` — show help via taskdog

## Usage Examples
- "taskdog --help"
- "taskdog <args>"
