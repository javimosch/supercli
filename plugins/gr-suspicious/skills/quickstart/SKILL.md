# Gr-suspicious Plugin

Suspicious lines of code detector for Git Repositories.

## Commands

### Operations
- `gr-suspicious repo scan` — scan repo via gr-suspicious
- `gr-suspicious file check` — check file via gr-suspicious
- `gr-suspicious commit check` — check commit via gr-suspicious
- `gr-suspicious branch check` — check branch via gr-suspicious
- `gr-suspicious history analyze` — analyze history via gr-suspicious
- `gr-suspicious rules list` — list rules via gr-suspicious
- `gr-suspicious config init` — init config via gr-suspicious
- `gr-suspicious config show` — show config via gr-suspicious
- `gr-suspicious report generate` — generate report via gr-suspicious
- `gr-suspicious output set` — set output via gr-suspicious
- `gr-suspicious verbose set` — set verbose via gr-suspicious

## Usage Examples
- "gr-suspicious --help"
- "gr-suspicious <args>"
