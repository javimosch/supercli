# Obelisk Plugin

Go package and CLI tool for saving web page as single HTML file.

## Commands

### Operations
- `obelisk page archive` — archive page via obelisk

## Usage Examples
- "obelisk --help"
- "obelisk <args>"

## Installation

```bash
go install github.com/go-shiori/obelisk/cmd/obelisk@latest
```

## Examples

```bash
obelisk --version
obelisk --help
```

## Key Features
- web\n- archive\n- html
