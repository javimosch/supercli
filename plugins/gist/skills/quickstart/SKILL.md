# Gist Plugin

Command line tool for creating and managing GitHub Gists.

## Commands

### Operations
- `gist gist create` — create gist via gist
- `gist gist list` — list gist via gist
- `gist gist delete` — delete gist via gist
- `gist gist edit` — edit gist via gist

## Usage Examples
- "gist --help"
- "gist <args>"

## Installation

```bash
go install github.com/babarot/gist@latest
```

## Examples

```bash
gist --version
gist --help
```
