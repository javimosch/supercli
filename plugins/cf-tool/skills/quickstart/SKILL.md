# Cf-tool Plugin

Command line tool for Codeforces competitive programming.

## Commands

### Operations
- `cf-tool contest parse` — parse contest via cf
- `cf-tool contest submit` — submit contest via cf

## Usage Examples
- "cf-tool --help"
- "cf-tool <args>"

## Installation

```bash
brew install cf-tool
```

## Examples

```bash
cf --version
cf --help
```
