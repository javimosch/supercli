# Kail Plugin

Kubernetes log viewer with pod filtering.

## Commands

### Operations
- `kail logs stream` — stream logs via kail

## Usage Examples
- "kail --help"
- "kail <args>"

## Installation

```bash
brew install boz/repo/kail
```

## Examples

```bash
kail --version
kail --help
```

## Key Features
- kubernetes\n- logs\n- monitoring
