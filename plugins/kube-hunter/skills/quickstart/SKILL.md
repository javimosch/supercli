# Kube-hunter Plugin

Hunt for security weaknesses in Kubernetes clusters.

## Commands

### Operations
- `kube-hunter cluster hunt` — hunt cluster via kube-hunter
- `kube-hunter cluster report` — report cluster via kube-hunter

## Usage Examples
- "kube-hunter --help"
- "kube-hunter <args>"

## Installation

```bash
brew install kube-hunter
```

## Examples

```bash
kube-hunter --version
kube-hunter --help
```

## Key Features
- kubernetes\n- security\n- penetration-testing
