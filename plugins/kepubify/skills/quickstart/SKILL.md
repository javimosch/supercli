# Kepubify Plugin

Fast ePub to Kobo ePub converter.

## Commands

### Operations
- `kepubify file convert` — convert file via kepubify

## Usage Examples
- "kepubify --help"
- "kepubify <args>"

## Installation

```bash
go install github.com/pgaskin/kepubify/cmd/kepubify@latest
```

## Examples

```bash
kepubify --version
kepubify --help
```

## Key Features
- epub\n- kobo\n- converter
