# Httpu Plugin

CLI for making HTTP requests with readable syntax.

## Commands

### Operations
- `httpu request send` — send request via httpu

## Usage Examples
- "httpu --help"
- "httpu <args>"

## Installation

```bash
go install github.com/hazbo/httpu@latest
```

## Examples

```bash
httpu --version
httpu --help
```

## Key Features
- http\n- client\n- api
