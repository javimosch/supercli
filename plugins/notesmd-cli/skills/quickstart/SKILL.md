# Notesmd-cli Plugin

Obsidian CLI (Community) - Interact with Obsidian in the terminal.

## Commands

### Operations
- `notesmd-cli note create` — create note via notesmd
- `notesmd-cli note search` — search note via notesmd
- `notesmd-cli note list` — list note via notesmd

## Usage Examples
- "notesmd-cli --help"
- "notesmd-cli <args>"

## Installation

```bash
go install github.com/Yakitrak/notesmd-cli@latest
```

## Examples

```bash
notesmd --version
notesmd --help
```

## Key Features
- notes\n- obsidian\n- markdown
