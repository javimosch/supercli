# K2tf Plugin

Convert Kubernetes YAML to Terraform HCL.

## Commands

### Operations
- `k2tf file convert` — convert file via k2tf

## Usage Examples
- "k2tf --help"
- "k2tf <args>"

## Installation

```bash
go install github.com/sl1pm4t/k2tf@latest
```

## Examples

```bash
k2tf --version
k2tf --help
```

## Key Features
- kubernetes\n- terraform\n- converter
