# Commit-and-tag-version Plugin

Automate versioning and CHANGELOG generation.

## Commands

### Operations
- `commit-and-tag-version version bump` — bump version via commit-and-tag-version

## Usage Examples
- "commit-and-tag-version --help"
- "commit-and-tag-version <args>"

## Installation

```bash
npm install -g commit-and-tag-version
```

## Examples

```bash
commit-and-tag-version --version
commit-and-tag-version --help
```

## Key Features
- versioning\n- changelog\n- git
