# Covermyass Plugin

Remove sensitive information from logs and files.

## Commands

### Operations
- `covermyass file clean` — clean file via covermyass

## Usage Examples
- "covermyass --help"
- "covermyass <args>"

## Installation

```bash
go install github.com/sundowndev/covermyass@latest
```

## Examples

```bash
covermyass --version
covermyass --help
```
