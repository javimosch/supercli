# Tfk8s Plugin

Convert Terraform HCL to Kubernetes YAML.

## Commands

### Operations
- `tfk8s file convert` — convert file via tfk8s

## Usage Examples
- "tfk8s --help"
- "tfk8s <args>"

## Installation

```bash
go install github.com/jrhouston/tfk8s@latest
```

## Examples

```bash
tfk8s --version
tfk8s --help
```

## Key Features
- terraform\n- kubernetes\n- converter
