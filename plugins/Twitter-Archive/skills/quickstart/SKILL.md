# Twitter-Archive Plugin

CLI tool to download media from bookmarked tweets.

## Commands

### Operations
- `Twitter-Archive media download` — download media via twitter-archive
- `Twitter-Archive tweet list` — list tweet via twitter-archive
- `Twitter-Archive bookmark fetch` — fetch bookmark via twitter-archive
- `Twitter-Archive output set` — set output via twitter-archive
- `Twitter-Archive format set` — set format via twitter-archive
- `Twitter-Archive verbose set` — set verbose via twitter-archive
- `Twitter-Archive help show` — show help via twitter-archive

## Usage Examples
- "Twitter-Archive --help"
- "Twitter-Archive <args>"

## Installation

```bash
pip install twitter-archive
```

## Examples

```bash
twitter-archive --version
twitter-archive --help
```

## Key Features
- twitter\n- media\n- downloader
