# Grpcmd Plugin

gRPC client for the terminal with readable syntax.

## Commands

### Operations
- `grpcmd request send` — send request via grpcmd

## Usage Examples
- "grpcmd --help"
- "grpcmd <args>"

## Installation

```bash
go install github.com/grpcmd/grpcmd@latest
```

## Examples

```bash
grpcmd --version
grpcmd --help
```

## Key Features
- grpc\n- api\n- client
