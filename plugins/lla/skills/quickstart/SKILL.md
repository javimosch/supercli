# Lla Plugin

Blazing fast ls replacement with superpowers written in Rust.

## Commands

### Operations
- `lla directory list` — list directory via lla
- `lla file show` — show file via lla
- `lla tree display` — display tree via lla
- `lla search run` — run search via lla
- `lla sort set` — set sort via lla
- `lla filter set` — set filter via lla
- `lla config init` — init config via lla
- `lla config show` — show config via lla
- `lla plugin list` — list plugin via lla
- `lla plugin install` — install plugin via lla
- `lla plugin remove` — remove plugin via lla

## Usage Examples
- "lla --help"
- "lla <args>"

## Installation

```bash
cargo install lla
```

## Examples

```bash
lla --version
```
