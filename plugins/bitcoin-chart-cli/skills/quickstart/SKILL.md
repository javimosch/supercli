# Bitcoin-chart-cli Plugin

Display Bitcoin price charts in the terminal.

## Commands

### Operations
- `bitcoin-chart-cli price show` — show price via bitcoin-chart

## Usage Examples
- "bitcoin-chart-cli --help"
- "bitcoin-chart-cli <args>"

## Installation

```bash
npm install -g bitcoin-chart-cli
```

## Examples

```bash
bitcoin-chart --version
bitcoin-chart --help
```

## Key Features
- bitcoin\n- crypto\n- chart
