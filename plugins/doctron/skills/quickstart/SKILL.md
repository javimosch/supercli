# Doctron Plugin

Convert Markdown and other documents to PDF or images.

## Commands

### Operations
- `doctron file convert` — convert file via doctron

## Usage Examples
- "doctron --help"
- "doctron <args>"

## Installation

```bash
go install github.com/lampnick/doctron@latest
```

## Examples

```bash
doctron --version
doctron --help
```

## Key Features
- pdf\n- markdown\n- converter
