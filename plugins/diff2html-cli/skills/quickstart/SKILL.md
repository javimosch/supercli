# Diff2html-cli Plugin

Pretty diff to HTML JavaScript CLI for converting diff output to beautiful HTML.

## Commands

### Operations
- `diff2html-cli diff convert` — convert diff via diff2html
- `diff2html-cli input set` — set input via diff2html
- `diff2html-cli output set` — set output via diff2html
- `diff2html-cli format set` — set format via diff2html
- `diff2html-cli style set` — set style via diff2html
- `diff2html-cli file compare` — compare file via diff2html
- `diff2html-cli directory compare` — compare directory via diff2html
- `diff2html-cli config show` — show config via diff2html
- `diff2html-cli verbose set` — set verbose via diff2html
- `diff2html-cli help show` — show help via diff2html

## Usage Examples
- "diff2html-cli --help"
- "diff2html-cli <args>"

## Installation

```bash
npm install -g diff2html-cli
```

## Examples

```bash
diff2html --version
diff2html --help
```

## Key Features
- diff\n- html\n- visualization
