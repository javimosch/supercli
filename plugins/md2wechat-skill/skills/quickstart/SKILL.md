# Md2wechat-skill Plugin

Convert Markdown to WeChat article format.

## Commands

### Operations
- `md2wechat-skill file convert` — convert file via md2wechat

## Usage Examples
- "md2wechat-skill --help"
- "md2wechat-skill <args>"

## Installation

```bash
go install github.com/geekjourneyx/md2wechat-skill@latest
```

## Examples

```bash
md2wechat --version
md2wechat --help
```

## Key Features
- markdown\n- wechat\n- converter
