# Circleci-cli Plugin

Command line tool for CircleCI.

## Commands

### Operations
- `circleci-cli config validate` — validate config via circleci
- `circleci-cli pipeline list` — list pipeline via circleci
- `circleci-cli orb list` — list orb via circleci

## Usage Examples
- "circleci-cli --help"
- "circleci-cli <args>"

## Installation

```bash
brew install circleci
```

## Examples

```bash
circleci --version
circleci --help
```
