# F2 Plugin

Cross-platform command-line tool for batch renaming files and directories quickly and safely.

## Commands

### Operations
- `f2 file rename` — rename file via f2

## Usage Examples
- "f2 --help"
- "f2 <args>"

## Installation

```bash
go install github.com/ayoisaiah/f2@latest
```

## Examples

```bash
f2 --version
f2 --help
```

## Key Features
- filesystem\n- rename\n- batch
