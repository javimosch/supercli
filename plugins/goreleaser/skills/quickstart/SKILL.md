# Goreleaser Plugin

Deliver Go binaries as fast and easily as possible.

## Commands

### Operations
- `goreleaser release run` — run release via goreleaser
- `goreleaser build run` — run build via goreleaser

## Usage Examples
- "goreleaser --help"
- "goreleaser <args>"

## Installation

```bash
brew install goreleaser
```

## Examples

```bash
goreleaser --version
goreleaser --help
```

## Key Features
- go\n- release\n- ci-cd
