# Mermaid-cli Plugin

Command line tool for the Mermaid diagramming language.

## Commands

### Operations
- `mermaid-cli file render` — render file via mmdc

## Usage Examples
- "mermaid-cli --help"
- "mermaid-cli <args>"

## Installation

```bash
npm install -g @mermaid-js/mermaid-cli
```

## Examples

```bash
mmdc --version
mmdc --help
```

## Key Features
- diagrams\n- charts\n- mermaid
