# Cssg-cli Plugin

CSS Generator for transforming user interfaces from design tokens.

## Commands

### Operations
- `cssg-cli tokens generate` — generate tokens via cssg
- `cssg-cli config init` — init config via cssg
- `cssg-cli config show` — show config via cssg
- `cssg-cli output set` — set output via cssg
- `cssg-cli format set` — set format via cssg
- `cssg-cli watch run` — run watch via cssg
- `cssg-cli build run` — run build via cssg
- `cssg-cli config show` — show config via cssg

## Usage Examples
- "cssg-cli --help"
- "cssg-cli <args>"

## Installation

```bash
npm install -g cssg-cli
```
