# Poutine Plugin

Supply chain vulnerability scanner for Go dependencies.

## Commands

### Operations
- `poutine repo scan` — scan repo via poutine

## Usage Examples
- "poutine --help"
- "poutine <args>"

## Installation

```bash
brew install poutine
```

## Examples

```bash
poutine --version
poutine --help
```

- security\n- scanner\n- supply-chain\n- go
