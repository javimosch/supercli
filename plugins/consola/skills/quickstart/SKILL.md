# Consola Plugin

Elegant Console Logger for Node.js and Browser with beautiful formatting.

## Commands

### Operations
- `consola log info` — info log via consola
- `consola log error` — error log via consola
- `consola log warn` — warn log via consola
- `consola log success` — success log via consola
- `consola log debug` — debug log via consola
- `consola log trace` — trace log via consola
- `consola log fatal` — fatal log via consola
- `consola log start` — start log via consola
- `consola log ready` — ready log via consola
- `consola config show` — show config via consola
- `consola verbose set` — set verbose via consola

## Usage Examples
- "consola --help"
- "consola <args>"

## Installation

```bash
npm install -g consola
```

## Examples

```bash
consola --version
consola --help
```
