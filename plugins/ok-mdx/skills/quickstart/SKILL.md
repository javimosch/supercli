# Ok-mdx Plugin

Zero-config MDX development and prototyping CLI.

## Commands

### Operations
- `ok-mdx mdx dev` — dev mdx via ok-mdx
- `ok-mdx mdx build` — build mdx via ok-mdx

## Usage Examples
- "ok-mdx --help"
- "ok-mdx <args>"

## Installation

```bash
npm install -g ok-mdx
```

## Examples

```bash
ok-mdx --version
ok-mdx --help
```

## Key Features
- mdx\n- react\n- prototyping
