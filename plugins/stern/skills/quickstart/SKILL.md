# Stern Plugin

Multi pod and container log tailing for Kubernetes.

## Commands

### Operations
- `stern pod logs` — logs pod via stern

## Usage Examples
- "stern --help"
- "stern <args>"

## Installation

```bash
brew install stern
```

## Examples

```bash
stern --version
stern --help
```

## Key Features
- kubernetes\n- k8s\n- logs
