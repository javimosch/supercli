# Butler Plugin

Swiss army knife CLI for managing development tasks.

## Commands

### Operations
- `butler task run` — run task via butler
- `butler task list` — list task via butler

## Usage Examples
- "butler --help"
- "butler <args>"

## Installation

```bash
go install github.com/mlabouardy/butler@latest
```

## Examples

```bash
butler --version
butler --help
```
