# Dagger Plugin

Programmable CI/CD engine that runs pipelines in containers.

## Commands

### Operations
- `dagger pipeline run` — run pipeline via dagger
- `dagger module list` — list module via dagger

## Usage Examples
- "dagger --help"
- "dagger <args>"

## Installation

```bash
curl -L https://dl.dagger.io/dagger/install.sh | sh
```

## Examples

```bash
dagger --version
dagger --help
```

## Key Features
- ci-cd\n- pipelines\n- containers
