# Dtctl Plugin

Command line tool for Dynatrace platform management.

## Commands

### Operations
- `dtctl config list` — list config via dtctl
- `dtctl tenant list` — list tenant via dtctl

## Usage Examples
- "dtctl --help"
- "dtctl <args>"

## Installation

```bash
go install github.com/dynatrace-oss/dtctl@latest
```

## Examples

```bash
dtctl --version
dtctl --help
```

## Key Features
- dynatrace\n- monitoring\n- observability
