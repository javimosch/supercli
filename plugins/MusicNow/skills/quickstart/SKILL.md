# MusicNow Plugin

CLI tool to download songs with metadata from various sources.

## Commands

### Operations
- `MusicNow song download` — download song via musicnow
- `MusicNow song search` — search song via musicnow
- `MusicNow album download` — download album via musicnow
- `MusicNow artist list` — list artist via musicnow
- `MusicNow format set` — set format via musicnow
- `MusicNow quality set` — set quality via musicnow
- `MusicNow output set` — set output via musicnow
- `MusicNow verbose set` — set verbose via musicnow
- `MusicNow help show` — show help via musicnow

## Usage Examples
- "MusicNow --help"
- "MusicNow <args>"

## Installation

```bash
pip install MusicNow
```

## Examples

```bash
musicnow --version
musicnow --help
```
