# Bpfd Plugin

Load and manage eBPF programs from user space.

## Commands

### Operations
- `bpfd program load` — load program via bpfd
- `bpfd program list` — list program via bpfd

## Usage Examples
- "bpfd --help"
- "bpfd <args>"

## Installation

```bash
go install github.com/genuinetools/bpfd@latest
```

## Examples

```bash
bpfd --version
bpfd --help
```

## Key Features
- ebpf\n- kernel\n- networking
