# Pingme Plugin

Unified CLI for sending notifications across multiple messaging platforms.

## Commands

### Operations
- `pingme message send` — send message via pingme
- `pingme slack send` — send slack via pingme
- `pingme telegram send` — send telegram via pingme
- `pingme discord send` — send discord via pingme
- `pingme teams send` — send teams via pingme
- `pingme email send` — send email via pingme
- `pingme pushover send` — send pushover via pingme
- `pingme config show` — show config via pingme

## Usage Examples
- "pingme --help"
- "pingme <args>"

## Installation

```bash
go install github.com/kha7iq/pingme@latest
```

pingme --version
