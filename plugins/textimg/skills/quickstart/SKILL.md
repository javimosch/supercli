# Textimg Plugin

Convert text to image from terminal.

## Commands

### Operations
- `textimg text convert` — convert text via textimg

## Usage Examples
- "textimg --help"
- "textimg <args>"

## Installation

```bash
go install github.com/jiro4989/textimg/cmd/textimg@latest
```

## Examples

```bash
textimg --version
textimg --help
```

## Key Features
- text\n- image\n- terminal
