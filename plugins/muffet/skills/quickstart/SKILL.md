# Muffet Plugin

Fast website link checker with recursion support.

## Commands

### Operations
- `muffet url check` — check url via muffet

## Usage Examples
- "muffet --help"
- "muffet <args>"

## Installation

```bash
brew install muffet
```

## Examples

```bash
muffet --version
muffet --help
```

## Key Features
- link-checker\n- web\n- crawler
