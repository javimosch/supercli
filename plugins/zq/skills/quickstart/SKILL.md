# Zq Plugin

Zed query language for structured data search and analytics.

## Commands

### Operations
- `zq data query` — query data via zq
- `zq data search` — search data via zq

## Usage Examples
- "zq --help"
- "zq <args>"

## Installation

```bash
brew install zed
```

## Examples

```bash
zq --version
zq --help
```

## Key Features
- zed\n- query\n- structured-data\n- analytics
