# Zamba Plugin

Python package for identifying hundreds of kinds of animals from camera trap videos and images.

## Commands

### Operations
- `zamba video identify` — identify video via zamba
- `zamba image identify` — identify image via zamba
- `zamba model train` — train model via zamba
- `zamba model list` — list model via zamba
- `zamba output set` — set output via zamba
- `zamba config show` — show config via zamba
- `zamba verbose set` — set verbose via zamba
- `zamba help show` — show help via zamba

## Usage Examples
- "zamba --help"
- "zamba <args>"

## Installation

```bash
pip install zamba
```
