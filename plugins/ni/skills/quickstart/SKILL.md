# Ni Plugin

Use the right package manager.

## Commands

### Operations
- `ni deps install` — install deps via ni
- `ni deps run` — run deps via ni
- `ni deps upgrade` — upgrade deps via ni

## Usage Examples
- "ni --help"
- "ni <args>"

## Installation

```bash
npm install -g @antfu/ni
```

## Examples

```bash
ni --version
ni --help
```
