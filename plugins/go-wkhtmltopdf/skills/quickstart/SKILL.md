# Go-wkhtmltopdf Plugin

Convert HTML to PDF using wkhtmltopdf in Go.

## Commands

### Operations
- `go-wkhtmltopdf file convert` — convert file via wkhtmltopdf

## Usage Examples
- "go-wkhtmltopdf --help"
- "go-wkhtmltopdf <args>"

## Installation

```bash
go install github.com/adrg/go-wkhtmltopdf/cmd/wkhtmltopdf@latest
```

## Examples

```bash
wkhtmltopdf --version
wkhtmltopdf --help
```

## Key Features
- html\n- pdf\n- converter
