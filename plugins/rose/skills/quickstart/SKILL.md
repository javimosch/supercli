# Rose Plugin

Command line tool for managing and viewing rose projects.

## Commands

### Operations
- `rose theme apply` — apply theme via rose
- `rose theme list` — list theme via rose

## Usage Examples
- "rose --help"
- "rose <args>"

## Installation

```bash
go install github.com/rose-pine/rose@latest
```

## Examples

```bash
rose --version
rose --help
```

- theme\n- rose-pine\n- cli
