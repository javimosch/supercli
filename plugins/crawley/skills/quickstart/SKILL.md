# Crawley Plugin

Unix-way web crawler and scraping tool.

## Commands

### Operations
- `crawley url crawl` — crawl url via crawley

## Usage Examples
- "crawley --help"
- "crawley <args>"

## Installation

```bash
go install github.com/s0rg/crawley/cmd/crawley@latest
```

## Examples

```bash
crawley --version
crawley --help
```

## Key Features
- web\n- crawler\n- scraping
