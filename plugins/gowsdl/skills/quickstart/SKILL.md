# Gowsdl Plugin

WSDL to Go code generator.

## Commands

### Operations
- `gowsdl wsdl generate` — generate wsdl via gowsdl

## Usage Examples
- "gowsdl --help"
- "gowsdl <args>"

## Installation

```bash
go install github.com/hooklift/gowsdl/cmd/gowsdl@latest
```

## Examples

```bash
gowsdl --version
gowsdl --help
```

## Key Features
- wsdl\n- soap\n- code-generation
