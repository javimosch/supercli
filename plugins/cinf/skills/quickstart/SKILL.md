# Cinf Plugin

Command line tool to view container resource usage.

## Commands

### Operations
- `cinf containers list` — list containers via cinf

## Usage Examples
- "cinf --help"
- "cinf <args>"

## Installation

```bash
go install github.com/mhausenblas/cinf@latest
```

## Examples

```bash
cinf --version
cinf --help
```

## Key Features
- containers\n- docker\n- resource-usage
