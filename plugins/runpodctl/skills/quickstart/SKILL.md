# Runpodctl Plugin

Command line tool for managing RunPod serverless GPUs.

## Commands

### Operations
- `runpodctl pod create` — create pod via runpodctl
- `runpodctl pod list` — list pod via runpodctl
- `runpodctl pod stop` — stop pod via runpodctl

## Usage Examples
- "runpodctl --help"
- "runpodctl <args>"

## Installation

```bash
brew install runpod/runpodctl/runpodctl
```

## Examples

```bash
runpodctl --version
runpodctl --help
```

## Key Features
- gpu\n- serverless\n- cloud
