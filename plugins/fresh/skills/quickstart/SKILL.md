# Fresh Plugin

Build and (re)start Go web apps after saving source files.

## Commands

### Operations
- `fresh watch start` — start watch via fresh

## Usage Examples
- "fresh --help"
- "fresh <args>"

## Installation

```bash
go install github.com/gravityblast/fresh@latest
```

## Examples

```bash
fresh --version
fresh --help
```

## Key Features
- live-reload\n- go\n- development
