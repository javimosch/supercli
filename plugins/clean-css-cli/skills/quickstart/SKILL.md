# Clean-css-cli Plugin

CSS minifier and optimizer CLI.

## Commands

### Operations
- `clean-css-cli file minify` — minify file via cleancss

## Usage Examples
- "clean-css-cli --help"
- "clean-css-cli <args>"

## Installation

```bash
npm install -g clean-css-cli
```

## Examples

```bash
cleancss --version
cleancss --help
```

## Key Features
- css\n- minifier\n- optimization
