# Asciigraph Plugin

Lightweight ASCII line charting with no dependencies.

## Commands

### Operations
- `asciigraph data chart` — chart data via asciigraph

## Usage Examples
- "asciigraph --help"
- "asciigraph <args>"

## Installation

```bash
go install github.com/guptarohit/asciigraph/cmd/asciigraph@latest
```

## Examples

```bash
asciigraph --version
asciigraph --help
```

## Key Features
- charts\n- ascii\n- visualization
