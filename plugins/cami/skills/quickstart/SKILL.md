# Cami Plugin

Lightweight CLI tool for configuration management with files.

## Commands

### Operations
- `cami config get` — get config via cami
- `cami config set` — set config via cami
- `cami config delete` — delete config via cami
- `cami config list` — list config via cami
- `cami file import` — import file via cami
- `cami file export` — export file via cami
- `cami template list` — list template via cami
- `cami template use` — use template via cami
- `cami config init` — init config via cami
- `cami config show` — show config via cami
- `cami verbose set` — set verbose via cami

## Usage Examples
- "cami --help"
- "cami <args>"

## Installation

```bash
npm install -g cami
```

## Examples

```bash
cami --version
cami --help
```
