# Protodep Plugin

Protocol Buffers dependency management tool.

## Commands

### Operations
- `protodep deps update` — update deps via protodep
- `protodep deps status` — status deps via protodep

## Usage Examples
- "protodep --help"
- "protodep <args>"

## Installation

```bash
go install github.com/stormcat24/protodep@latest
```

## Examples

```bash
protodep --version
protodep --help
```

- protobuf\n- dependencies\n- grpc
