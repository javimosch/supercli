# Kubeval Plugin

Validate Kubernetes configuration files.

## Commands

### Operations
- `kubeval manifest validate` — validate manifest via kubeval

## Usage Examples
- "kubeval --help"
- "kubeval <args>"

## Installation

```bash
brew install kubeval
```

## Examples

```bash
kubeval --version
kubeval --help
```

## Key Features
- kubernetes\n- validation\n- yaml
