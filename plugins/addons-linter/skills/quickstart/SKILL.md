# Addons-linter Plugin

Linter for Firefox WebExtensions and addons.

## Commands

### Operations
- `addons-linter extension lint` — lint extension via addons-linter

## Usage Examples
- "addons-linter --help"
- "addons-linter <args>"

## Installation

```bash
npm install -g addons-linter
```

## Examples

```bash
addons-linter --version
addons-linter --help
```

## Key Features
- firefox\n- webextensions\n- linting
