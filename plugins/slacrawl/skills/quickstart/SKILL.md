# Slacrawl Plugin

Crawl and archive Slack workspaces from the command line.

## Commands

### Operations
- `slacrawl workspace crawl` — crawl workspace via slacrawl
- `slacrawl channel list` — list channel via slacrawl

## Usage Examples
- "slacrawl --help"
- "slacrawl <args>"

## Installation

```bash
go install github.com/vincentkoc/slacrawl@latest
```

## Examples

```bash
slacrawl --version
slacrawl --help
```

## Key Features
- slack\n- crawler\n- archive
