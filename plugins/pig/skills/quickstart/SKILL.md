# Pig Plugin

PostgreSQL extension package manager CLI.

## Commands

### Operations
- `pig extension install` — install extension via pig
- `pig extension list` — list extension via pig
- `pig extension remove` — remove extension via pig
- `pig extension update` — update extension via pig

## Usage Examples
- "pig --help"
- "pig <args>"

## Installation

```bash
brew install pgsty/pig/pig
```

## Examples

```bash
pig --version
pig --help
```

- postgresql\n- extensions\n- database
