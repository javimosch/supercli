# Rawkit Plugin

Development toolkit and project scaffolding CLI.

## Commands

### Operations
- `rawkit project scaffold` — scaffold project via rawkit
- `rawkit project init` — init project via rawkit

## Usage Examples
- "rawkit --help"
- "rawkit <args>"

## Installation

```bash
npm install -g rawkit
```

## Examples

```bash
rawkit --version
rawkit --help
```

## Key Features
- development\n- toolkit\n- scaffold
