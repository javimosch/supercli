# Next-build-id Plugin

Generate unique build IDs for Next.js applications.

## Commands

### Operations
- `next-build-id build generate` — generate build via next-build-id

## Usage Examples
- "next-build-id --help"
- "next-build-id <args>"

## Installation

```bash
npm install -g next-build-id
```

## Examples

```bash
next-build-id --version
next-build-id --help
```

## Key Features
- nextjs\n- build\n- versioning
