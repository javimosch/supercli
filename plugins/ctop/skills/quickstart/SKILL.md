# Ctop Plugin

Top-like interface for container metrics.

## Commands

### Operations
- `ctop containers list` — list containers via ctop

## Usage Examples
- "ctop --help"
- "ctop <args>"

## Installation

```bash
brew install ctop
```

## Examples

```bash
ctop --version
ctop --help
```

## Key Features
- containers\n- docker\n- monitoring
