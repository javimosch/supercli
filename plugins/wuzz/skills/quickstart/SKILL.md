# Wuzz Plugin

Interactive HTTP inspection and testing tool.

## Commands

### Operations
- `wuzz request test` — test request via wuzz

## Usage Examples
- "wuzz --help"
- "wuzz <args>"

## Installation

```bash
brew install wuzz
```

## Examples

```bash
wuzz --version
wuzz --help
```

## Key Features
- http\n- testing\n- api
