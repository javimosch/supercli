# MyJWT Plugin

CLI for cracking testing vulnerabilities on Json Web Token JWT security testing.

## Commands

### Operations
- `MyJWT token decode` — decode token via myjwt
- `MyJWT token validate` — validate token via myjwt
- `MyJWT token crack` — crack token via myjwt
- `MyJWT algorithm test` — test algorithm via myjwt
- `MyJWT payload show` — show payload via myjwt
- `MyJWT header show` — show header via myjwt
- `MyJWT verbose set` — set verbose via myjwt
- `MyJWT help show` — show help via myjwt

## Usage Examples
- "MyJWT --help"
- "MyJWT <args>"
