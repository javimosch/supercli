# Cdncheck Plugin

A utility to detect various technology for a given IP address.

## Commands

### Operations
- `cdncheck ip check` — check ip via cdncheck

## Usage Examples
- "cdncheck --help"
- "cdncheck <args>"

## Installation

```bash
go install github.com/projectdiscovery/cdncheck/cmd/cdncheck@latest
```

## Examples

```bash
cdncheck --version
cdncheck --help
```
