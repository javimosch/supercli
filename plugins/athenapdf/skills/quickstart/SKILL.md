# Athenapdf Plugin

Convert HTML to PDF using Chrome headless.

## Commands

### Operations
- `athenapdf file convert` — convert file via athenapdf

## Usage Examples
- "athenapdf --help"
- "athenapdf <args>"

## Installation

```bash
go install github.com/arachnys/athenapdf/cmd/athenapdf@latest
```

## Examples

```bash
athenapdf --version
athenapdf --help
```

## Key Features
- pdf\n- html\n- headless
