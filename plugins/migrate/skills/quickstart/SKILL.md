# Migrate Plugin

Database migration tool with CLI.

## Commands

### Operations
- `migrate migration up` — up migration via migrate
- `migrate migration down` — down migration via migrate
- `migrate migration create` — create migration via migrate

## Usage Examples
- "migrate --help"
- "migrate <args>"

## Installation

```bash
brew install golang-migrate
```

## Examples

```bash
migrate --version
migrate --help
```

## Key Features
- database\n- migration\n- sql
