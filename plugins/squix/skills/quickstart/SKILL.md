# Squix Plugin

Run SQL queries directly against CSV, JSON, Parquet files.

## Commands

### Operations
- `squix query execute` — execute query via squix

## Usage Examples
- "squix --help"
- "squix <args>"

## Installation

```bash
go install github.com/eduardofuncao/squix@latest
```

## Examples

```bash
squix --version
squix --help
```

## Key Features
- sql\n- csv\n- json\n- database
