# Gitkit-js Plugin

JavaScript toolkit for Git operations.

## Commands

### Operations
- `gitkit-js git run` — run git via gitkit

## Usage Examples
- "gitkit-js --help"
- "gitkit-js <args>"

## Installation

```bash
npm install -g gitkit
```

## Examples

```bash
gitkit --version
gitkit --help
```

## Key Features
- git\n- toolkit\n- cli
