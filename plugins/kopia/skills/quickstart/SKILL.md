# Kopia Plugin

Cross-platform backup tool with fast, incremental backups.

## Commands

### Operations
- `kopia snapshot list` — list snapshot via kopia
- `kopia repository connect` — connect repository via kopia

## Usage Examples
- "kopia --help"
- "kopia <args>"

## Installation

```bash
brew install kopia
```

## Examples

```bash
kopia --version
kopia --help
```

## Key Features
- backup\n- archive\n- encryption
