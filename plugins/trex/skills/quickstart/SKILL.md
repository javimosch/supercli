# Trex Plugin

Command line tool for extracting and processing text data.

## Commands

### Operations
- `trex text extract` — extract text via trex
- `trex pattern search` — search pattern via trex

## Usage Examples
- "trex --help"
- "trex <args>"

## Installation

```bash
go install github.com/samyakbardiya/trex@latest
```

## Examples

```bash
trex --version
trex --help
```
