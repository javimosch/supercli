# Srgn Plugin

A grep-like tool which understands source code syntax for manipulation and search.

## Commands

### Operations
- `srgn code search` — search code via srgn
- `srgn code replace` — replace code via srgn
- `srgn language set` — set language via srgn
- `srgn pattern set` — set pattern via srgn
- `srgn case set` — set case via srgn
- `srgn regex set` — set regex via srgn
- `srgn multiline set` — set multiline via srgn
- `srgn output save` — save output via srgn
- `srgn config show` — show config via srgn

## Usage Examples
- "srgn --help"
- "srgn <args>"

## Installation

```bash
cargo install srgn
```

## Examples

```bash
srgn --version
srgn --help
```
