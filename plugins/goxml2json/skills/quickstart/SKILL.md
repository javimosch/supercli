# Goxml2json Plugin

Convert XML to JSON with a simple Go CLI.

## Commands

### Operations
- `goxml2json file convert` — convert file via goxml2json

## Usage Examples
- "goxml2json --help"
- "goxml2json <args>"

## Installation

```bash
go install github.com/basgys/goxml2json/cmd/goxml2json@latest
```

## Examples

```bash
goxml2json --version
goxml2json --help
```

## Key Features
- xml\n- json\n- converter
