# Is-esm Plugin

Check if a package is an ES module.

## Commands

### Operations
- `is-esm package check` — check package via is-esm

## Usage Examples
- "is-esm --help"
- "is-esm <args>"

## Installation

```bash
npm install -g is-esm
```

## Examples

```bash
is-esm --version
is-esm --help
```

## Key Features
- esm\n- javascript\n- module-check
