# Apizza Plugin

Order pizza from the command line.

## Commands

### Operations
- `apizza order place` — place order via apizza
- `apizza menu show` — show menu via apizza
- `apizza history show` — show history via apizza

## Usage Examples
- "apizza --help"
- "apizza <args>"

## Installation

```bash
go install github.com/harrybrwn/apizza@latest
```

## Examples

```bash
apizza --version
apizza --help
```
