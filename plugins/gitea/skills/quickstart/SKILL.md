# Gitea Plugin

Painless self-hosted Git service with CLI administration.

## Commands

### Operations
- `gitea admin create-user` — create-user admin via gitea
- `gitea admin delete-user` — delete-user admin via gitea
- `gitea migrate list` — list migrate via gitea
- `gitea dump create` — create dump via gitea

## Usage Examples
- "gitea --help"
- "gitea <args>"

## Installation

```bash
brew install gitea
```

## Examples

```bash
gitea --version
gitea --help
```

## Key Features
- git\n- self-hosted\n- scm
