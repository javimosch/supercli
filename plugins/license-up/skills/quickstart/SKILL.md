# License-up Plugin

License generator and manager for open-source projects.

## Commands

### Operations
- `license-up license generate` — generate license via license-up

## Usage Examples
- "license-up --help"
- "license-up <args>"

## Installation

```bash
npm install -g license-up
```

## Examples

```bash
license-up --version
license-up --help
```

- license\n- generator\n- open-source
