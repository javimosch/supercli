# Aq Plugin

Query AWS resources with SQL for easy data extraction and analysis.

## Commands

### Operations
- `aq resource query` — query resource via aq
- `aq resource list` — list resource via aq
- `aq service set` — set service via aq
- `aq output set` — set output via aq
- `aq format set` — set format via aq
- `aq config show` — show config via aq
- `aq verbose set` — set verbose via aq
- `aq help show` — show help via aq

## Usage Examples
- "aq --help"
- "aq <args>"

## Installation

```bash
pip install aq
```

## Examples

```bash
aq --version
aq --help
```
