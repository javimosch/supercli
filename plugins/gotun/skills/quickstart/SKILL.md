# Gotun Plugin

Create VPN tunnels with ease.

## Commands

### Operations
- `gotun tunnel create` — create tunnel via gotun
- `gotun tunnel list` — list tunnel via gotun
- `gotun tunnel destroy` — destroy tunnel via gotun

## Usage Examples
- "gotun --help"
- "gotun <args>"

## Installation

```bash
go install github.com/Sesame2/gotun@latest
```

## Examples

```bash
gotun --version
gotun --help
```

## Key Features
- vpn\n- tunnel\n- networking
