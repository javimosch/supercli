# Modd Plugin

Flexible developer tool for responding to filesystem changes.

## Commands

### Operations
- `modd watch start` — start watch via modd

## Usage Examples
- "modd --help"
- "modd <args>"

## Installation

```bash
brew install modd
```

## Examples

```bash
modd --version
modd --help
```
