# Iponmap Plugin

Locate IP addresses on a map in terminal.

## Commands

### Operations
- `iponmap ip locate` — locate ip via iponmap

## Usage Examples
- "iponmap --help"
- "iponmap <args>"

## Installation

```bash
npm install -g iponmap
```

## Examples

```bash
iponmap --version
iponmap --help
```

## Key Features
- ip\n- geolocation\n- map
