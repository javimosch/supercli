# Ghosts Plugin

A hosts file manager for developers.

## Commands

### Operations
- `ghosts hosts check` — check hosts via ghosts

## Usage Examples
- "ghosts --help"
- "ghosts <args>"

## Installation

```bash
go install github.com/StevenBlack/ghosts@latest
```

## Examples

```bash
ghosts --version
ghosts --help
```

## Key Features
- hosts\n- dns\n- networking
