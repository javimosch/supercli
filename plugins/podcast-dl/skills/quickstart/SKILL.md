# Podcast-dl Plugin

Download podcast episodes from RSS feeds.

## Commands

### Operations
- `podcast-dl feed download` — download feed via podcast-dl

## Usage Examples
- "podcast-dl --help"
- "podcast-dl <args>"

## Installation

```bash
npm install -g podcast-dl
```

## Examples

```bash
podcast-dl --version
podcast-dl --help
```

## Key Features
- podcast\n- download\n- rss
