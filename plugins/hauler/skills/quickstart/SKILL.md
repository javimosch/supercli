# Hauler Plugin

Airgap Swiss Army Knife for content management.

## Commands

### Operations
- `hauler store add` — add store via hauler
- `hauler store info` — info store via hauler
- `hauler store serve` — serve store via hauler
- `hauler store extract` — extract store via hauler
- `hauler store load` — load store via hauler

## Usage Examples
- "hauler --help"
- "hauler <args>"

## Installation

```bash
brew install hauler-dev/tap/hauler
```

## Examples

```bash
hauler --version
hauler --help
```
