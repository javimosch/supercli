# Sonoscli Plugin

Command line controller for Sonos speakers.

## Commands

### Operations
- `sonoscli speaker discover` — discover speaker via sonos
- `sonoscli speaker play` — play speaker via sonos
- `sonoscli speaker pause` — pause speaker via sonos
- `sonoscli speaker volume` — volume speaker via sonos

## Usage Examples
- "sonoscli --help"
- "sonoscli <args>"

## Installation

```bash
go install github.com/steipete/sonoscli@latest
```

## Examples

```bash
sonos --version
sonos --help
```
