# Dnspyre Plugin

CLI tool for a high QPS DNS benchmark.

## Commands

### Operations
- `dnspyre server benchmark` — benchmark server via dnspyre

## Usage Examples
- "dnspyre --help"
- "dnspyre <args>"

## Installation

```bash
brew install dnspyre
```

## Examples

```bash
dnspyre --version
dnspyre --help
```

## Key Features
- dns\n- benchmark\n- network
