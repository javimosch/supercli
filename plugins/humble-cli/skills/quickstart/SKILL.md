# Humble-cli Plugin

Command line tool for Humble Bundle purchases and downloads.

## Commands

### Operations
- `humble-cli library list` — list library via humble
- `humble-cli bundle show` — show bundle via humble
- `humble-cli bundle download` — download bundle via humble

## Usage Examples
- "humble-cli --help"
- "humble-cli <args>"

## Installation

```bash
cargo install humble-cli
```

## Examples

```bash
humble --version
humble --help
```

## Key Features
- humble-bundle\n- games\n- download
