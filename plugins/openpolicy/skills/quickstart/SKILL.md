# Openpolicy Plugin

Framework for building privacy-first applications from the command line.

## Commands

### Operations
- `openpolicy policy init` — init policy via openpolicy
- `openpolicy policy validate` — validate policy via openpolicy
- `openpolicy policy list` — list policy via openpolicy
- `openpolicy policy apply` — apply policy via openpolicy
- `openpolicy audit run` — run audit via openpolicy
- `openpolicy report generate` — generate report via openpolicy
- `openpolicy config show` — show config via openpolicy

## Usage Examples
- "openpolicy --help"
- "openpolicy <args>"

## Installation

```bash
npm install -g openpolicy
```

## Examples

```bash
openpolicy --version
openpolicy --help
```
