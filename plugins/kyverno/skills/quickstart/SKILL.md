# Kyverno Plugin

Kubernetes native policy management CLI.

## Commands

### Operations
- `kyverno policy test` — test policy via kyverno
- `kyverno policy apply` — apply policy via kyverno
- `kyverno policy scan` — scan policy via kyverno

## Usage Examples
- "kyverno --help"
- "kyverno <args>"

## Installation

```bash
brew install kyverno
```

## Examples

```bash
kyverno --version
kyverno --help
```
