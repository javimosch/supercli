# Dupl Plugin

Tool for finding code clones in Go projects.

## Commands

### Operations
- `dupl code check` — check code via dupl

## Usage Examples
- "dupl --help"
- "dupl <args>"

## Installation

```bash
go install github.com/mibk/dupl@latest
```

dupl --version
