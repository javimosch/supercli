# Npm-upgrade Plugin

Interactive CLI utility to upgrade npm packages.

## Commands

### Operations
- `npm-upgrade deps check` — check deps via npm-upgrade
- `npm-upgrade deps upgrade` — upgrade deps via npm-upgrade

## Usage Examples
- "npm-upgrade --help"
- "npm-upgrade <args>"

## Installation

```bash
npm install -g npm-upgrade
```

## Examples

```bash
npm-upgrade --version
npm-upgrade --help
```

## Key Features
- npm\n- upgrade\n- dependencies
