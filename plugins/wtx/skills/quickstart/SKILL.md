# Wtx Plugin

Collection of transport implementations and tools focused on web technologies.

## Commands

### Operations
- `wtx request send` — send request via wtx
- `wtx response show` — show response via wtx
- `wtx server start` — start server via wtx
- `wtx client connect` — connect client via wtx
- `wtx benchmark run` — run benchmark via wtx
- `wtx config show` — show config via wtx

## Usage Examples
- "wtx --help"
- "wtx <args>"

## Installation

```bash
cargo install wtx
```

## Examples

```bash
wtx --version
wtx --help
```
