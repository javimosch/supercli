# Code-radio-cli Plugin

Command line music radio client for freeCodeCamp Code Radio.

## Commands

### Operations
- `code-radio-cli radio play` — play radio via code-radio
- `code-radio-cli radio pause` — pause radio via code-radio
- `code-radio-cli radio next` — next radio via code-radio
- `code-radio-cli radio prev` — prev radio via code-radio
- `code-radio-cli volume set` — set volume via code-radio
- `code-radio-cli station list` — list station via code-radio
- `code-radio-cli config show` — show config via code-radio

## Usage Examples
- "code-radio-cli --help"
- "code-radio-cli <args>"

## Installation

```bash
cargo install code-radio-cli
```

## Examples

```bash
code-radio --version
```
