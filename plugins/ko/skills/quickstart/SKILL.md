# Ko Plugin

Build and deploy Go containers.

## Commands

### Operations
- `ko build run` — run build via ko

## Usage Examples
- "ko --help"
- "ko <args>"

## Installation

```bash
go install github.com/google/ko@latest
```

## Examples

```bash
ko --version
ko --help
```

## Key Features
- go\n- containers\n- kubernetes
