# Imagineui Plugin

CLI tool to generate wireframes from a localized human-readable format.

## Commands

### Operations
- `imagineui design generate` — generate design via imagineui
- `imagineui wireframe create` — create wireframe via imagineui
- `imagineui output save` — save output via imagineui
- `imagineui format set` — set format via imagineui
- `imagineui template list` — list template via imagineui
- `imagineui config show` — show config via imagineui

## Usage Examples
- "imagineui --help"
- "imagineui <args>"

## Installation

```bash
npm install -g imagineui
```

## Examples

```bash
imagineui --version
imagineui --help
```
