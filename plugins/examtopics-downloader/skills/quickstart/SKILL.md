# Examtopics-downloader Plugin

Download exam content from ExamTopics website.

## Commands

### Operations
- `examtopics-downloader exam download` — download exam via examtopics-downloader
- `examtopics-downloader topic list` — list topic via examtopics-downloader

## Usage Examples
- "examtopics-downloader --help"
- "examtopics-downloader <args>"

## Installation

```bash
go install github.com/thatonecodes/examtopics-downloader@latest
```

## Examples

```bash
examtopics-downloader --version
examtopics-downloader --help
```
