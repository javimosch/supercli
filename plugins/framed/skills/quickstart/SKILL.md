# Framed Plugin

Architect your project with effortless files and directories management.

## Commands

### Operations
- `framed project scaffold` — scaffold project via framed
- `framed template list` — list template via framed
- `framed template use` — use template via framed

## Usage Examples
- "framed --help"
- "framed <args>"

## Installation

```bash
go install github.com/mactat/framed@latest
```

## Examples

```bash
framed --version
framed --help
```

## Key Features
- project-scaffold\n- file-management\n- generator
