# Mail-deduplicate Plugin

CLI to deduplicate mails from mail boxes to reduce clutter.

## Commands

### Operations
- `mail-deduplicate mail scan` — scan mail via mail-deduplicate
- `mail-deduplicate mail deduplicate` — deduplicate mail via mail-deduplicate
- `mail-deduplicate mail remove` — remove mail via mail-deduplicate
- `mail-deduplicate mailbox set` — set mailbox via mail-deduplicate
- `mail-deduplicate output set` — set output via mail-deduplicate
- `mail-deduplicate verbose set` — set verbose via mail-deduplicate
- `mail-deduplicate help show` — show help via mail-deduplicate

## Usage Examples
- "mail-deduplicate --help"
- "mail-deduplicate <args>"

## Installation

pip install mail-deduplicate
