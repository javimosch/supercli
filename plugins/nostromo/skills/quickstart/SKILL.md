# Nostromo Plugin

CLI for building powerful aliases and command shortcuts.

## Commands

### Operations
- `nostromo alias add` — add alias via nostromo
- `nostromo alias list` — list alias via nostromo
- `nostromo alias remove` — remove alias via nostromo
- `nostromo alias sync` — sync alias via nostromo

## Usage Examples
- "nostromo --help"
- "nostromo <args>"

## Installation

```bash
brew install pokanop/tap/nostromo
```

## Examples

```bash
nostromo --version
nostromo --help
```
