# Nitrolaunch Plugin

Supercharged Minecraft launcher for managing Minecraft instances.

## Commands

### Operations
- `nitrolaunch instance list` — list instance via nitrolaunch
- `nitrolaunch instance create` — create instance via nitrolaunch
- `nitrolaunch instance launch` — launch instance via nitrolaunch
- `nitrolaunch instance delete` — delete instance via nitrolaunch
- `nitrolaunch mod install` — install mod via nitrolaunch
- `nitrolaunch mod remove` — remove mod via nitrolaunch
- `nitrolaunch mod list` — list mod via nitrolaunch
- `nitrolaunch config show` — show config via nitrolaunch
- `nitrolaunch version list` — list version via nitrolaunch
- `nitrolaunch version install` — install version via nitrolaunch

## Usage Examples
- "nitrolaunch --help"
- "nitrolaunch <args>"

## Installation

```bash
cargo install nitrolaunch
```

## Examples

```bash
nitrolaunch --version
```
