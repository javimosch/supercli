# Http-server-rs Plugin

Simple and configurable command-line HTTP server written in Rust.

## Commands

### Operations
- `http-server-rs server start` — start server via http-server
- `http-server-rs config show` — show config via http-server
- `http-server-rs port set` — set port via http-server
- `http-server-rs directory serve` — serve directory via http-server
- `http-server-rs tls enable` — enable tls via http-server
- `http-server-rs cors enable` — enable cors via http-server

## Usage Examples
- "http-server-rs --help"
- "http-server-rs <args>"

## Installation

```bash
cargo install http-server
```

## Examples

```bash
http-server --version
http-server --help
```
