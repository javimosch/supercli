# Evernote2md Plugin

Convert Evernote ENEX files to Markdown.

## Commands

### Operations
- `evernote2md file convert` — convert file via evernote2md

## Usage Examples
- "evernote2md --help"
- "evernote2md <args>"

## Installation

```bash
go install github.com/wormi4ok/evernote2md@latest
```

## Examples

```bash
evernote2md --version
evernote2md --help
```

## Key Features
- evernote\n- markdown\n- converter
