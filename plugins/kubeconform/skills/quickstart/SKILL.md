# Kubeconform Plugin

Fast Kubernetes manifest validation with custom schemas.

## Commands

### Operations
- `kubeconform manifest validate` — validate manifest via kubeconform

## Usage Examples
- "kubeconform --help"
- "kubeconform <args>"

## Installation

```bash
brew install kubeconform
```

## Examples

```bash
kubeconform --version
kubeconform --help
```

## Key Features
- kubernetes\n- validation\n- manifest
