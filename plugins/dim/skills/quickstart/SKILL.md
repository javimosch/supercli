# Dim Plugin

Manage open data in your project like a package manager.

## Commands

### Operations
- `dim data install` — install data via dim
- `dim data update` — update data via dim
- `dim data list` — list data via dim
- `dim data remove` — remove data via dim
- `dim data search` — search data via dim
- `dim data info` — info data via dim
- `dim data import` — import data via dim
- `dim data export` — export data via dim
- `dim config show` — show config via dim
- `dim registry list` — list registry via dim

## Usage Examples
- "dim --help"
- "dim <args>"

## Installation

```bash
npm install -g dim
```

## Examples

```bash
dim --version
dim --help
```
