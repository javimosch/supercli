# Yajsv Plugin

Yet Another JSON Schema Validator CLI.

## Commands

### Operations
- `yajsv file validate` — validate file via yajsv

## Usage Examples
- "yajsv --help"
- "yajsv <args>"

## Installation

```bash
go install github.com/neilpa/yajsv@latest
```

## Examples

```bash
yajsv --version
yajsv --help
```

- json\n- validation\n- schema
