# Twitch-cli Plugin

Official Twitch CLI for developing on Twitch platform.

## Commands

### Operations
- `twitch-cli api request` — request api via twitch
- `twitch-cli event trigger` — trigger event via twitch
- `twitch-cli event list` — list event via twitch
- `twitch-cli configure setup` — setup configure via twitch
- `twitch-cli mock-api start` — start mock-api via twitch
- `twitch-cli token get` — get token via twitch

## Usage Examples
- "twitch-cli --help"
- "twitch-cli <args>"

## Installation

```bash
brew install twitchdev/twitch/twitch-cli
```

```bash
twitch --version
```
