# Ain Plugin

HTTP API client for the terminal.

## Commands

### Operations
- `ain request execute` — execute request via ain

## Usage Examples
- "ain --help"
- "ain <args>"

## Installation

```bash
brew install ain
```

## Examples

```bash
ain --version
ain --help
```

## Key Features
- http\n- api\n- client
