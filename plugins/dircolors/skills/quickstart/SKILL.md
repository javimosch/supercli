# Dircolors Plugin

Colorize ls output with Nord theme support.

## Commands

### Operations
- `dircolors dir colorize` — colorize dir via dircolors

## Usage Examples
- "dircolors --help"
- "dircolors <args>"

## Installation

```bash
npm install -g dircolors
```

## Examples

```bash
dircolors --version
dircolors --help
```

## Key Features
- ls\n- colors\n- terminal
