# Klog Plugin

Command-line tool for time tracking in a human-readable, plain-text file format.

## Commands

### Operations
- `klog time track` — track time via klog
- `klog report generate` — generate report via klog

## Usage Examples
- "klog --help"
- "klog <args>"

## Installation

```bash
brew install klog
```

## Examples

```bash
klog --version
klog --help
```

## Key Features
- time-tracking\n- productivity\n- plaintext
