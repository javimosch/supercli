# Xo Plugin

Command line utility that composes regular expression matches.

## Commands

### Operations
- `xo pattern match` — match pattern via xo
- `xo pattern replace` — replace pattern via xo

## Usage Examples
- "xo --help"
- "xo <args>"

## Installation

```bash
go install github.com/ezekg/xo@latest
```

## Examples

```bash
xo --version
xo --help
```
