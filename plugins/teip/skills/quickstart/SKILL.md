# Teip Plugin

Masking tape to help commands do one thing well with selective processing.

## Commands

### Operations
- `teip input filter` — filter input via teip
- `teip pattern select` — select pattern via teip
- `teip range select` — select range via teip
- `teip field select` — select field via teip
- `teip delimiter set` — set delimiter via teip
- `teip regex select` — select regex via teip
- `teip ignore-case set` — set ignore-case via teip
- `teip reverse set` — set reverse via teip
- `teip config show` — show config via teip

## Usage Examples
- "teip --help"
- "teip <args>"

## Installation

```bash
cargo install teip
```
