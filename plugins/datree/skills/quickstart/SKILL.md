# Datree Plugin

Prevent Kubernetes misconfigurations from reaching production.

## Commands

### Operations
- `datree config test` — test config via datree
- `datree policy list` — list policy via datree

## Usage Examples
- "datree --help"
- "datree <args>"

## Installation

```bash
brew install datree
```

## Examples

```bash
datree --version
datree --help
```

## Key Features
- kubernetes\n- policy\n- validation
