# Lev Plugin

LevelDB CLI tool for database inspection.

## Commands

### Operations
- `lev db inspect` — inspect db via lev

## Usage Examples
- "lev --help"
- "lev <args>"

## Installation

```bash
npm install -g lev
```

## Examples

```bash
lev --version
lev --help
```
