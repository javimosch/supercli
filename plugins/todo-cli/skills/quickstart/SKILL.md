# Todo-cli Plugin

Command-line tool to manage Todo lists with simple and efficient interface.

## Commands

### Operations
- `todo-cli todo add` — add todo via todo-cli
- `todo-cli todo list` — list todo via todo-cli
- `todo-cli todo remove` — remove todo via todo-cli
- `todo-cli todo complete` — complete todo via todo-cli
- `todo-cli todo edit` — edit todo via todo-cli
- `todo-cli list show` — show list via todo-cli
- `todo-cli config show` — show config via todo-cli
- `todo-cli verbose set` — set verbose via todo-cli
- `todo-cli help show` — show help via todo-cli

## Usage Examples
- "todo-cli --help"
- "todo-cli <args>"

## Installation

```bash
pip install todo-cli
```
