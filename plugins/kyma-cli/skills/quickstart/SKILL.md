# Kyma-cli Plugin

CLI for deploying and managing Kyma runtime.

## Commands

### Operations
- `kyma-cli cluster provision` — provision cluster via kyma
- `kyma-cli cluster deploy` — deploy cluster via kyma
- `kyma-cli function init` — init function via kyma
- `kyma-cli function apply` — apply function via kyma

## Usage Examples
- "kyma-cli --help"
- "kyma-cli <args>"

## Installation

```bash
brew install kyma-cli
```

## Examples

```bash
kyma --version
kyma --help
```
