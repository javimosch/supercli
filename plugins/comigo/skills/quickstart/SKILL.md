# Comigo Plugin

Comic reader and server for CBR, CBZ, EPUB, and manga.

## Commands

### Operations
- `comigo file serve` — serve file via comigo

## Usage Examples
- "comigo --help"
- "comigo <args>"

## Installation

```bash
go install github.com/yumenaka/comigo@latest
```

## Examples

```bash
comigo --version
comigo --help
```

## Key Features
- comics\n- reader\n- manga\n- epub
