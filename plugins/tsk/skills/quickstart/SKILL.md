# Tsk Plugin

Simple todo list manager for the terminal.

## Commands

### Operations
- `tsk task add` — add task via tsk
- `tsk task list` — list task via tsk
- `tsk task done` — done task via tsk

## Usage Examples
- "tsk --help"
- "tsk <args>"

## Installation

```bash
go install github.com/kakengloh/tsk@latest
```

## Examples

```bash
tsk --version
tsk --help
```

## Key Features
- todo\n- task\n- cli
