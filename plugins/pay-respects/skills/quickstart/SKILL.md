# Pay-respects Plugin

Command suggestions command-not-found and thefuck replacement written in Rust.

## Commands

### Operations
- `pay-respects command fix` — fix command via pay-respects
- `pay-respects command suggest` — suggest command via pay-respects
- `pay-respects rule add` — add rule via pay-respects
- `pay-respects rule list` — list rule via pay-respects
- `pay-respects config init` — init config via pay-respects
- `pay-respects config show` — show config via pay-respects

## Usage Examples
- "pay-respects --help"
- "pay-respects <args>"

## Installation

```bash
cargo install pay-respects
```
