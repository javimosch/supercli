# Color-generate Plugin

Generate and manipulate color palettes from the command line.

## Commands

### Operations
- `color-generate color generate` — generate color via color-generate

## Usage Examples
- "color-generate --help"
- "color-generate <args>"

## Installation

```bash
npm install -g color-generate
```

## Examples

```bash
color-generate --version
color-generate --help
```
