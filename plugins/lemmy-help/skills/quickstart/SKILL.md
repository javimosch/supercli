# Lemmy-help Plugin

CLI to generate vim and neovim help documentation from EmmyLua annotations.

## Commands

### Operations
- `lemmy-help doc generate` — generate doc via lemmy-help
- `lemmy-help file parse` — parse file via lemmy-help
- `lemmy-help config show` — show config via lemmy-help
- `lemmy-help output save` — save output via lemmy-help

## Usage Examples
- "lemmy-help --help"
- "lemmy-help <args>"

## Installation

```bash
cargo install lemmy-help
```

## Examples

```bash
lemmy-help --version
lemmy-help --help
```

## Key Features
- vim\n- neovim\n- documentation
