# Bsky Plugin

Bluesky CLI client for posting and interacting.

## Commands

### Operations
- `bsky post create` — create post via bsky
- `bsky timeline show` — show timeline via bsky
- `bsky user show` — show user via bsky

## Usage Examples
- "bsky --help"
- "bsky <args>"

## Installation

```bash
go install github.com/mattn/bsky@latest
```

## Examples

```bash
bsky --version
bsky --help
```

- bluesky\n- social\n- cli
