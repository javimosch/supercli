# Sub2sing-box Plugin

Convert proxy subscription links to sing-box configuration.

## Commands

### Operations
- `sub2sing-box subscription convert` — convert subscription via sub2sing-box

## Usage Examples
- "sub2sing-box --help"
- "sub2sing-box <args>"

## Installation

```bash
go install github.com/bestnite/sub2sing-box@latest
```

## Examples

```bash
sub2sing-box --version
sub2sing-box --help
```

## Key Features
- proxy\n- sing-box\n- converter
