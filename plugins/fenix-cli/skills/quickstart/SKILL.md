# Fenix-cli Plugin

Command line interface for Fenix code search engine.

## Commands

### Operations
- `fenix-cli search run` — run search via fenix
- `fenix-cli index build` — build index via fenix

## Usage Examples
- "fenix-cli --help"
- "fenix-cli <args>"

## Installation

```bash
go install github.com/fenixsoft/fenix-cli@latest
```

## Examples

```bash
fenix --version
fenix --help
```
