# Goplaces Plugin

Modern Google Places command line tool in Go.

## Commands

### Operations
- `goplaces place search` — search place via goplaces
- `goplaces place details` — details place via goplaces
- `goplaces nearby find` — find nearby via goplaces

## Usage Examples
- "goplaces --help"
- "goplaces <args>"

## Installation

```bash
go install github.com/steipete/goplaces@latest
```

## Examples

```bash
goplaces --version
goplaces --help
```
