# Typeorm-fixtures Plugin

Fixtures loader for TypeORM to load test data into your database.

## Commands

### Operations
- `typeorm-fixtures fixtures load` — load fixtures via typeorm-fixtures
- `typeorm-fixtures fixtures sync` — sync fixtures via typeorm-fixtures
- `typeorm-fixtures fixtures resolve` — resolve fixtures via typeorm-fixtures
- `typeorm-fixtures config set` — set config via typeorm-fixtures
- `typeorm-fixtures connection set` — set connection via typeorm-fixtures
- `typeorm-fixtures output set` — set output via typeorm-fixtures
- `typeorm-fixtures config show` — show config via typeorm-fixtures
- `typeorm-fixtures verbose set` — set verbose via typeorm-fixtures
- `typeorm-fixtures help show` — show help via typeorm-fixtures

## Usage Examples
- "typeorm-fixtures --help"
- "typeorm-fixtures <args>"

## Installation

```bash
npm install -g typeorm-fixtures-cli
```

```bash
typeorm-fixtures --version
```
