# Xc Plugin

Markdown task runner that executes code blocks.

## Commands

### Operations
- `xc task run` — run task via xc
- `xc task list` — list task via xc

## Usage Examples
- "xc --help"
- "xc <args>"

## Installation

```bash
go install github.com/joerdav/xc@latest
```

## Examples

```bash
xc --version
xc --help
```

## Key Features
- markdown\n- task-runner\n- documentation
