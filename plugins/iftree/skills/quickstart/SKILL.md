# Iftree Plugin

Display network interfaces as a tree structure.

## Commands

### Operations
- `iftree interfaces show` — show interfaces via iftree
- `iftree interfaces tree` — tree interfaces via iftree

## Usage Examples
- "iftree --help"
- "iftree <args>"

## Installation

```bash
go install github.com/t1anz0ng/iftree@latest
```

## Examples

```bash
iftree --version
iftree --help
```

## Key Features
- network\n- interfaces\n- diagnostics
