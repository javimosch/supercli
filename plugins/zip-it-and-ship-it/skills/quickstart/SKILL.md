# Zip-it-and-ship-it Plugin

Create deployable function bundles for Netlify and AWS Lambda.

## Commands

### Operations
- `zip-it-and-ship-it functions bundle` — bundle functions via zip-it-and-ship-it
- `zip-it-and-ship-it functions list` — list functions via zip-it-and-ship-it

## Usage Examples
- "zip-it-and-ship-it --help"
- "zip-it-and-ship-it <args>"

## Installation

```bash
npm install -g @netlify/zip-it-and-ship-it
```

## Examples

```bash
zip-it-and-ship-it --version
zip-it-and-ship-it --help
```

## Key Features
- serverless\n- bundler\n- deployment
