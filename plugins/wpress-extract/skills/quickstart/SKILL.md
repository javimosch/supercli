# Wpress-extract Plugin

Extract .wpress backup files from the command line.

## Commands

### Operations
- `wpress-extract file extract` — extract file via wpress-extract

## Usage Examples
- "wpress-extract --help"
- "wpress-extract <args>"

## Installation

```bash
npm install -g wpress-extract
```

## Examples

```bash
wpress-extract --version
wpress-extract --help
```

## Key Features
- wordpress\n- backup\n- extract
