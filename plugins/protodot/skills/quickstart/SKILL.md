# Protodot Plugin

Generate DOT language visualization from Protocol Buffer files.

## Commands

### Operations
- `protodot proto convert` — convert proto via protodot

## Usage Examples
- "protodot --help"
- "protodot <args>"

## Installation

```bash
go install github.com/seamia/protodot/cmd/protodot@latest
```

## Examples

```bash
protodot --version
protodot --help
```

## Key Features
- protobuf\n- dot\n- visualization
