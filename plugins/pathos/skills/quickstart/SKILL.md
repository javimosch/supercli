# Pathos Plugin

Path manipulation and file system utility CLI.

## Commands

### Operations
- `pathos path resolve` — resolve path via pathos
- `pathos path normalize` — normalize path via pathos
- `pathos path join` — join path via pathos

## Usage Examples
- "pathos --help"
- "pathos <args>"

## Installation

```bash
go install github.com/chip/pathos@latest
```

## Examples

```bash
pathos --version
pathos --help
```
