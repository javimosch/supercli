# Task Plugin

Task runner / build tool that aims to be simpler than GNU Make.

## Commands

### Operations
- `task task run` — run task via task
- `task task list` — list task via task
- `task task init` — init task via task

## Usage Examples
- "task --help"
- "task <args>"

## Installation

```bash
brew install go-task/tap/go-task
```

## Examples

```bash
task --version
task --help
```

## Key Features
- task-runner\n- build-tool\n- makefile
