# Yamldiff Plugin

CLI tool to diff two YAML files with structural awareness.

## Commands

### Operations
- `yamldiff files diff` — diff files via yamldiff
- `yamldiff files compare` — compare files via yamldiff

## Usage Examples
- "yamldiff --help"
- "yamldiff <args>"

## Installation

```bash
go install github.com/sahilm/yamldiff@latest
```

## Examples

```bash
yamldiff --version
yamldiff --help
```

## Key Features
- yaml\n- diff\n- configuration
