# Kbld Plugin

Seamlessly incorporates image building and image pushing into your development and deployment workflows.

## Commands

### Operations
- `kbld image resolve` — resolve image via kbld

## Usage Examples
- "kbld --help"
- "kbld <args>"

## Installation

```bash
brew install carvel-dev/tap/kbld
```

## Examples

```bash
kbld --version
kbld --help
```

## Key Features
- kubernetes\n- containers\n- devops
