# Flavours Plugin

Easy to use base16 scheme manager that integrates with any workflow.

## Commands

### Operations
- `flavours scheme apply` — apply scheme via flavours
- `flavours scheme list` — list scheme via flavours
- `flavours scheme info` — info scheme via flavours
- `flavours scheme update` — update scheme via flavours
- `flavours config init` — init config via flavours
- `flavours config show` — show config via flavours
- `flavours template apply` — apply template via flavours

## Usage Examples
- "flavours --help"
- "flavours <args>"

## Installation

```bash
cargo install flavours
```

## Examples

```bash
flavours --version
flavours --help
```
