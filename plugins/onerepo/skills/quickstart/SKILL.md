# Onerepo Plugin

Easy strict safe and fast JavaScript and TypeScript monorepo toolchain.

## Commands

### Operations
- `onerepo repo init` — init repo via onerepo
- `onerepo repo verify` — verify repo via onerepo
- `onerepo repo build` — build repo via onerepo
- `onerepo repo test` — test repo via onerepo
- `onerepo repo lint` — lint repo via onerepo
- `onerepo repo format` — format repo via onerepo
- `onerepo repo graph` — graph repo via onerepo
- `onerepo workspace add` — add workspace via onerepo
- `onerepo workspace list` — list workspace via onerepo
- `onerepo workspace remove` — remove workspace via onerepo
- `onerepo plugin install` — install plugin via onerepo
- `onerepo plugin list` — list plugin via onerepo
- `onerepo config show` — show config via onerepo
- `onerepo help show` — show help via onerepo

## Usage Examples
- "onerepo --help"
- "onerepo <args>"
