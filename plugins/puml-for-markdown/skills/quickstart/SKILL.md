# Puml-for-markdown Plugin

Generate PlantUML diagrams for Markdown files.

## Commands

### Operations
- `puml-for-markdown file generate` — generate file via puml-for-markdown

## Usage Examples
- "puml-for-markdown --help"
- "puml-for-markdown <args>"

## Installation

```bash
npm install -g puml-for-markdown
```

## Examples

```bash
puml-for-markdown --version
puml-for-markdown --help
```

## Key Features
- plantuml\n- markdown\n- diagrams
