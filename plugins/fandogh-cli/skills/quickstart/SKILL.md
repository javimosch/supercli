# Fandogh-cli Plugin

Fandogh CLI for managing cloud services and deployments.

## Commands

### Operations
- `fandogh-cli service deploy` — deploy service via fandogh
- `fandogh-cli service list` — list service via fandogh
- `fandogh-cli service logs` — logs service via fandogh
- `fandogh-cli service delete` — delete service via fandogh
- `fandogh-cli image build` — build image via fandogh
- `fandogh-cli config show` — show config via fandogh
- `fandogh-cli verbose set` — set verbose via fandogh
- `fandogh-cli help show` — show help via fandogh

## Usage Examples
- "fandogh-cli --help"
- "fandogh-cli <args>"

## Installation

```bash
pip install fandogh-cli
```
