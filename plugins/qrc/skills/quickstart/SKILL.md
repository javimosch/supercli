# Qrc Plugin

QR code generator for text terminals with ASCII art and Sixel support.

## Commands

### Operations
- `qrc text encode` — encode text via qrc
- `qrc file encode` — encode file via qrc
- `qrc url encode` — encode url via qrc
- `qrc binary encode` — encode binary via qrc

## Usage Examples
- "qrc --help"
- "qrc <args>"

## Installation

```bash
go install github.com/fumiyas/qrc@latest
```

## Examples

```bash
qrc --version
qrc --help
```
