# Havener Plugin

Swiss army knife for Kubernetes cluster operations.

## Commands

### Operations
- `havener node top` — top node via havener
- `havener pod exec` — exec pod via havener

## Usage Examples
- "havener --help"
- "havener <args>"

## Installation

```bash
brew install havener
```

## Examples

```bash
havener --version
havener --help
```

## Key Features
- kubernetes\n- cluster\n- devops
