# Fclones Plugin

Efficient duplicate file finder.

## Commands

### Operations
- `fclones duplicates find` — find duplicates via fclones
- `fclones group list` — list group via fclones

## Usage Examples
- "fclones --help"
- "fclones <args>"

## Installation

```bash
cargo install fclones
```

## Examples

```bash
fclones --version
fclones --help
```

## Key Features
- filesystem\n- duplicates\n- cleanup
