# Supervizer Plugin

Process manager and supervisor CLI.

## Commands

### Operations
- `supervizer process manage` — manage process via supervizer

## Usage Examples
- "supervizer --help"
- "supervizer <args>"

## Installation

```bash
npm install -g supervizer
```

## Examples

```bash
supervizer --version
supervizer --help
```

## Key Features
- process\n- manager\n- supervisor
