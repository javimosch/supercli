# Dschnp Plugin

Custom search engines via CLI with defaultList feature for quick lookups.

## Commands

### Operations
- `dschnp search run` — run search via dschnp
- `dschnp engine list` — list engine via dschnp
- `dschnp engine add` — add engine via dschnp
- `dschnp engine remove` — remove engine via dschnp
- `dschnp default set` — set default via dschnp
- `dschnp config show` — show config via dschnp
- `dschnp query run` — run query via dschnp
- `dschnp open run` — run open via dschnp

## Usage Examples
- "dschnp --help"
- "dschnp <args>"

## Installation

```bash
npm install -g dschnp
```
