# Ramadan-cli Plugin

CLI to check Sehar and Iftar times in Ramadan anywhere in the world.

## Commands

### Operations
- `ramadan-cli ramadan times` — times ramadan via ramadan-cli
- `ramadan-cli sehri show` — show sehri via ramadan-cli
- `ramadan-cli iftar show` — show iftar via ramadan-cli
- `ramadan-cli location set` — set location via ramadan-cli
- `ramadan-cli date set` — set date via ramadan-cli
- `ramadan-cli calendar show` — show calendar via ramadan-cli
- `ramadan-cli config show` — show config via ramadan-cli
- `ramadan-cli verbose set` — set verbose via ramadan-cli

## Usage Examples
- "ramadan-cli --help"
- "ramadan-cli <args>"

## Installation

```bash
npm install -g ramadan-cli
```
