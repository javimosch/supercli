# Curd Plugin

Anime episode manager and scraper CLI.

## Commands

### Operations
- `curd episode manage` — manage episode via curd

## Usage Examples
- "curd --help"
- "curd <args>"

## Installation

```bash
go install github.com/Wraient/curd@latest
```

```bash
curd --version
curd --help
```
