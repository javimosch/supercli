# I18next-scanner Plugin

Scan code and extract i18next translation keys.

## Commands

### Operations
- `i18next-scanner code scan` — scan code via i18next-scanner

## Usage Examples
- "i18next-scanner --help"
- "i18next-scanner <args>"

## Installation

```bash
npm install -g i18next-scanner
```

## Examples

```bash
i18next-scanner --version
i18next-scanner --help
```

## Key Features
- i18n\n- translation\n- scanner
