# Dbin Plugin

Poor man's package manager with statically linked binaries.

## Commands

### Operations
- `dbin package install` — install package via dbin
- `dbin package remove` — remove package via dbin
- `dbin package search` — search package via dbin
- `dbin package list` — list package via dbin
- `dbin package update` — update package via dbin
- `dbin package info` — info package via dbin
- `dbin run exec` — exec run via dbin
- `dbin package sync` — sync package via dbin

## Usage Examples
- "dbin --help"
- "dbin <args>"

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/xplshn/dbin/main/install | sh
```
