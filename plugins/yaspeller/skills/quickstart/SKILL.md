# Yaspeller Plugin

Spell checker for text and Markdown files.

## Commands

### Operations
- `yaspeller file check` — check file via yaspeller

## Usage Examples
- "yaspeller --help"
- "yaspeller <args>"

## Installation

```bash
npm install -g yaspeller
```

## Examples

```bash
yaspeller --version
yaspeller --help
```

## Key Features
- spelling\n- language\n- text
