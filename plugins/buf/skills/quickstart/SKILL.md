# Buf Plugin

Protocol Buffers tool for linting and breaking change detection.

## Commands

### Operations
- `buf proto lint` — lint proto via buf
- `buf proto generate` — generate proto via buf
- `buf proto breaking` — breaking proto via buf

## Usage Examples
- "buf --help"
- "buf <args>"

## Installation

```bash
brew install buf
```

## Examples

```bash
buf --version
buf --help
```

- protobuf\n- lint\n- api
