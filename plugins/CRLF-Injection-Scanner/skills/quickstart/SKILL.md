# CRLF-Injection-Scanner Plugin

Command line tool for testing CRLF injection on a list of domains.

## Commands

### Operations
- `CRLF-Injection-Scanner domain scan` — scan domain via crlf-scanner
- `CRLF-Injection-Scanner domain list` — list domain via crlf-scanner
- `CRLF-Injection-Scanner file load` — load file via crlf-scanner
- `CRLF-Injection-Scanner output set` — set output via crlf-scanner
- `CRLF-Injection-Scanner verbose set` — set verbose via crlf-scanner
- `CRLF-Injection-Scanner help show` — show help via crlf-scanner

## Usage Examples
- "CRLF-Injection-Scanner --help"
- "CRLF-Injection-Scanner <args>"

## Installation

```bash
pip install crlf-scanner
```
