# Go-aws-sso Plugin

AWS SSO credential helper for command line access.

## Commands

### Operations
- `go-aws-sso account list` — list account via go-aws-sso
- `go-aws-sso config setup` — setup config via go-aws-sso
- `go-aws-sso login run` — run login via go-aws-sso

## Usage Examples
- "go-aws-sso --help"
- "go-aws-sso <args>"

## Installation

```bash
go install github.com/theurichde/go-aws-sso@latest
```

## Examples

```bash
go-aws-sso --version
go-aws-sso --help
```
