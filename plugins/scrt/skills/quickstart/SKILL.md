# Scrt Plugin

Password manager and secret store CLI tool.

## Commands

### Operations
- `scrt secret get` — get secret via scrt
- `scrt secret set` — set secret via scrt

## Usage Examples
- "scrt --help"
- "scrt <args>"

## Installation

```bash
brew install scrt
```

## Examples

```bash
scrt --version
scrt --help
```

## Key Features
- password\n- secrets\n- encryption
