# Cli-kenai Plugin

CLI tools for JavaScript projects with code generation and scaffolding.

## Commands

### Operations
- `cli-kenai project scaffold` — scaffold project via kenai
- `cli-kenai component generate` — generate component via kenai
- `cli-kenai config init` — init config via kenai
- `cli-kenai config show` — show config via kenai
- `cli-kenai template list` — list template via kenai
- `cli-kenai template add` — add template via kenai

## Usage Examples
- "cli-kenai --help"
- "cli-kenai <args>"

## Installation

```bash
npm install -g cli-kenai
```

## Examples

```bash
kenai --version
kenai --help
```
