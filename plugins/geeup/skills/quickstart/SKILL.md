# Geeup Plugin

Simple CLI for Google Earth Engine Uploads to manage cloud assets.

## Commands

### Operations
- `geeup asset upload` — upload asset via geeup
- `geeup asset list` — list asset via geeup
- `geeup asset delete` — delete asset via geeup
- `geeup task status` — status task via geeup
- `geeup config show` — show config via geeup
- `geeup verbose set` — set verbose via geeup
- `geeup help show` — show help via geeup

## Usage Examples
- "geeup --help"
- "geeup <args>"

## Installation

```bash
pip install geeup
```

## Examples

```bash
geeup --version
```
