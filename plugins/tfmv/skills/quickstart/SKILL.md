# Tfmv Plugin

CLI to rename Terraform resources and generate moved blocks.

## Commands

### Operations
- `tfmv resource rename` — rename resource via tfmv

## Usage Examples
- "tfmv --help"
- "tfmv <args>"

## Installation

```bash
brew install suzuki-shunsuke/tap/tfmv
```

## Examples

```bash
tfmv --version
tfmv --help
```

## Key Features
- terraform\n- infrastructure\n- refactoring
