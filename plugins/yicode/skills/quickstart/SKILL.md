# Yicode Plugin

CLI tool for code generation and management.

## Commands

### Operations
- `yicode code generate` — generate code via yicode
- `yicode project init` — init project via yicode

## Usage Examples
- "yicode --help"
- "yicode <args>"

## Installation

```bash
npm install -g yicode
```

## Examples

```bash
yicode --version
yicode --help
```

- code-generation\n- development\n- cli
