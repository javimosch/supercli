# Homebutler Plugin

Home automation and device management CLI.

## Commands

### Operations
- `homebutler device list` — list device via homebutler
- `homebutler device control` — control device via homebutler
- `homebutler scene run` — run scene via homebutler

## Usage Examples
- "homebutler --help"
- "homebutler <args>"

## Installation

```bash
go install github.com/Higangssh/homebutler@latest
```

## Examples

```bash
homebutler --version
homebutler --help
```
