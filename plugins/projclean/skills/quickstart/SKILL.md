# Projclean Plugin

Project dependencies and build artifacts cleanup tool.

## Commands

### Operations
- `projclean project clean` — clean project via projclean
- `projclean artifact list` — list artifact via projclean
- `projclean dependency analyze` — analyze dependency via projclean
- `projclean config show` — show config via projclean

## Usage Examples
- "projclean --help"
- "projclean <args>"

## Installation

```bash
cargo install projclean
```

## Examples

```bash
projclean --version
projclean --help
```
