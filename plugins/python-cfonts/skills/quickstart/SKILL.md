# Python-cfonts Plugin

Sexy fonts for the console with ASCII art and styling.

## Commands

### Operations
- `python-cfonts text display` — display text via cfonts
- `python-cfonts font set` — set font via cfonts
- `python-cfonts color set` — set color via cfonts
- `python-cfonts style set` — set style via cfonts
- `python-cfonts align set` — set align via cfonts
- `python-cfonts verbose set` — set verbose via cfonts
- `python-cfonts help show` — show help via cfonts

## Usage Examples
- "python-cfonts --help"
- "python-cfonts <args>"
