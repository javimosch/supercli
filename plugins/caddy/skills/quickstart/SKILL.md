# Caddy Plugin

Fast and extensible multi-platform HTTP/2 web server.

## Commands

### Operations
- `caddy config adapt` — adapt config via caddy
- `caddy server run` — run server via caddy
- `caddy reverse-proxy start` — start reverse-proxy via caddy

## Usage Examples
- "caddy --help"
- "caddy <args>"

## Installation

```bash
brew install caddy
```

## Examples

```bash
caddy --version
caddy --help
```

## Key Features
- web-server\n- https\n- proxy
