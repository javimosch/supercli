# Certok Plugin

Command line tool to check the validity and expiration dates of SSL certificates.

## Commands

### Operations
- `certok cert check` — check cert via certok

## Usage Examples
- "certok --help"
- "certok <args>"

## Installation

```bash
go install github.com/genuinetools/certok@latest
```

## Examples

```bash
certok --version
certok --help
```

## Key Features
- ssl\n- certificates\n- security
