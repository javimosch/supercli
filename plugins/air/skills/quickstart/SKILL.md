# Air Plugin

Live reload for Go apps in development.

## Commands

### Operations
- `air dev start` — start dev via air
- `air init init` — init init via air

## Usage Examples
- "air --help"
- "air <args>"

## Installation

```bash
go install github.com/cosmtrek/air@latest
```

## Examples

```bash
air --version
air --help
```

- live-reload\n- go\n- development
