# Epub-utils Plugin

Python library and CLI tool for inspecting ePub from the terminal.

## Commands

### Operations
- `epub-utils epub info` — info epub via epub-utils
- `epub-utils epub list` — list epub via epub-utils
- `epub-utils epub extract` — extract epub via epub-utils
- `epub-utils metadata show` — show metadata via epub-utils
- `epub-utils toc show` — show toc via epub-utils
- `epub-utils output set` — set output via epub-utils
- `epub-utils verbose set` — set verbose via epub-utils
- `epub-utils help show` — show help via epub-utils

## Usage Examples
- "epub-utils --help"
- "epub-utils <args>"
