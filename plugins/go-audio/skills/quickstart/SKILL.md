# Go-audio Plugin

Offline solution to convert PDFs into audiobooks.

## Commands

### Operations
- `go-audio pdf convert` — convert pdf via go-audio
- `go-audio text speak` — speak text via go-audio

## Usage Examples
- "go-audio --help"
- "go-audio <args>"

## Installation

```bash
go install github.com/Harry-027/go-audio@latest
```

## Examples

```bash
go-audio --version
go-audio --help
```

## Key Features
- pdf\n- audiobook\n- text-to-speech
