# Texture-compressor Plugin

CLI tool for texture compression using ASTC, ETC, PVRTC and S3TC in a KTX container.

## Commands

### Operations
- `texture-compressor image compress` — compress image via texture-compressor
- `texture-compressor format set` — set format via texture-compressor
- `texture-compressor quality set` — set quality via texture-compressor
- `texture-compressor input set` — set input via texture-compressor
- `texture-compressor output set` — set output via texture-compressor
- `texture-compressor batch run` — run batch via texture-compressor
- `texture-compressor config show` — show config via texture-compressor
- `texture-compressor verbose set` — set verbose via texture-compressor

## Usage Examples
- "texture-compressor --help"
- "texture-compressor <args>"
