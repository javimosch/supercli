# Tfsort Plugin

A CLI utility to sort Terraform variables and outputs.

## Commands

### Operations
- `tfsort file sort` — sort file via tfsort

## Usage Examples
- "tfsort --help"
- "tfsort <args>"

## Installation

```bash
brew install tfsort
```

## Examples

```bash
tfsort --version
tfsort --help
```

## Key Features
- terraform\n- infrastructure\n- sorting
