# Fhirbase Plugin

FHIR healthcare data CLI for PostgreSQL.

## Commands

### Operations
- `fhirbase data query` — query data via fhirbase

## Usage Examples
- "fhirbase --help"
- "fhirbase <args>"

## Installation

```bash
go install github.com/fhirbase/fhirbase@latest
```

## Examples

```bash
fhirbase --version
fhirbase --help
```

## Key Features
- healthcare\n- fhir\n- postgresql
