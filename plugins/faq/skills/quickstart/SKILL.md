# Faq Plugin

Format agnostic querier for structured data like JSON, XML, CSV.

## Commands

### Operations
- `faq data query` — query data via faq

## Usage Examples
- "faq --help"
- "faq <args>"

## Installation

```bash
brew install jzelinskie/faq/faq
```

## Examples

```bash
faq --version
faq --help
```

## Key Features
- json\n- xml\n- csv\n- query
