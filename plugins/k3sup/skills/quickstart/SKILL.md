# K3sup Plugin

Bootstrap k3s clusters over SSH with one command.

## Commands

### Operations
- `k3sup cluster install` — install cluster via k3sup
- `k3sup app install` — install app via k3sup

## Usage Examples
- "k3sup --help"
- "k3sup <args>"

## Installation

```bash
brew install k3sup
```

## Examples

```bash
k3sup --version
k3sup --help
```

## Key Features
- kubernetes\n- k3s\n- cluster
