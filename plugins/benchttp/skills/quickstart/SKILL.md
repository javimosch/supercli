# Benchttp Plugin

HTTP benchmarking tool with configurable requests.

## Commands

### Operations
- `benchttp request bench` — bench request via benchttp
- `benchttp config show` — show config via benchttp

## Usage Examples
- "benchttp --help"
- "benchttp <args>"

## Installation

```bash
go install github.com/siadat/benchttp@latest
```

## Examples

```bash
benchttp --version
benchttp --help
```

## Key Features
- benchmark\n- http\n- testing
