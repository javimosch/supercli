# Appfairy Plugin

Web application development and deployment CLI tool.

## Commands

### Operations
- `Appfairy app build` — build app via appfairy
- `Appfairy app deploy` — deploy app via appfairy

## Usage Examples
- "Appfairy --help"
- "Appfairy <args>"

## Installation

```bash
npm install -g appfairy
```

## Examples

```bash
appfairy --version
appfairy --help
```
