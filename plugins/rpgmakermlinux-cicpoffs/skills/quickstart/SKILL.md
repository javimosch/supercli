# Rpgmakermlinux-cicpoffs Plugin

RPG Maker MV/MZ Linux compatibility tool.

## Commands

### Operations
- `rpgmakermlinux-cicpoffs game run` — run game via rpgmakermlinux

## Usage Examples
- "rpgmakermlinux-cicpoffs --help"
- "rpgmakermlinux-cicpoffs <args>"

## Installation

```bash
npm install -g rpgmakermlinux-cicpoffs
```

rpgmakermlinux --version
