# Hopp-cli Plugin

HTTP CLI client for Hoppscotch API testing.

## Commands

### Operations
- `hopp-cli request send` — send request via hopp
- `hopp-cli collection run` — run collection via hopp
- `hopp-cli environment list` — list environment via hopp

## Usage Examples
- "hopp-cli --help"
- "hopp-cli <args>"

## Installation

```bash
go install github.com/hoppscotch/hopp-cli@latest
```

## Examples

```bash
hopp --version
hopp --help
```

## Key Features
- http\n- api-testing\n- cli
