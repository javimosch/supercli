# Slack-cli Plugin

Create develop and deploy Slack apps from the command line.

## Commands

### Operations
- `slack-cli app create` — create app via slack
- `slack-cli app deploy` — deploy app via slack
- `slack-cli function create` — create function via slack
- `slack-cli function deploy` — deploy function via slack
- `slack-cli function run` — run function via slack
- `slack-cli trigger create` — create trigger via slack
- `slack-cli manifest info` — info manifest via slack
- `slack-cli manifest validate` — validate manifest via slack
- `slack-cli activity list` — list activity via slack

## Usage Examples
- "slack-cli --help"
- "slack-cli <args>"

## Installation

```bash
brew install slack
```
