# Percollate Plugin

Turn web pages into PDF, EPUB, or HTML books.

## Commands

### Operations
- `percollate page convert` — convert page via percollate

## Usage Examples
- "percollate --help"
- "percollate <args>"

## Installation

```bash
npm install -g percollate
```

## Examples

```bash
percollate --version
percollate --help
```

## Key Features
- pdf\n- epub\n- html\n- web
