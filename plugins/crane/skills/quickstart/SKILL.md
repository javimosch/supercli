# Crane Plugin

Tool for interacting with remote container registries.

## Commands

### Operations
- `crane image pull` — pull image via crane
- `crane image push` — push image via crane
- `crane image ls` — ls image via crane
- `crane image digest` — digest image via crane

## Usage Examples
- "crane --help"
- "crane <args>"

## Installation

```bash
brew install crane
```

## Examples

```bash
crane --version
crane --help
```

## Key Features
- containers\n- registry\n- oci
