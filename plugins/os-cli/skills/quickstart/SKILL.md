# Os-cli Plugin

Fine-grained cross-platform OS information from the command line.

## Commands

### Operations
- `os-cli os info` — info os via os-cli
- `os-cli platform show` — show platform via os-cli
- `os-cli arch show` — show arch via os-cli
- `os-cli cpu show` — show cpu via os-cli
- `os-cli memory show` — show memory via os-cli
- `os-cli uptime show` — show uptime via os-cli
- `os-cli user show` — show user via os-cli
- `os-cli hostname show` — show hostname via os-cli
- `os-cli release show` — show release via os-cli
- `os-cli config show` — show config via os-cli

## Usage Examples
- "os-cli --help"
- "os-cli <args>"

## Installation

```bash
npm install -g os-cli
```

## Examples

```bash
os-cli --version
os-cli --help
```
