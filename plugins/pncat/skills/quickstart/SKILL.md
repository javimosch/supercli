# Pncat Plugin

Enhanced catalogs feature with PNPM, Yarn, Bun and Vlt support.

## Commands

### Operations
- `pncat catalog list` — list catalog via pncat
- `pncat catalog create` — create catalog via pncat
- `pncat catalog update` — update catalog via pncat
- `pncat catalog delete` — delete catalog via pncat
- `pncat package add` — add package via pncat
- `pncat package remove` — remove package via pncat
- `pncat sync run` — run sync via pncat
- `pncat config show` — show config via pncat

## Usage Examples
- "pncat --help"
- "pncat <args>"

## Installation

```bash
npm install -g pncat
```

```bash
pncat --version
```
