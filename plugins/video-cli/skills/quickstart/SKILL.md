# Video-cli Plugin

Command line tools for quick video editing and processing.

## Commands

### Operations
- `video-cli video trim` — trim video via video-cli
- `video-cli video merge` — merge video via video-cli
- `video-cli video convert` — convert video via video-cli
- `video-cli audio extract` — extract audio via video-cli
- `video-cli subtitle add` — add subtitle via video-cli
- `video-cli output set` — set output via video-cli
- `video-cli config show` — show config via video-cli
- `video-cli verbose set` — set verbose via video-cli
- `video-cli help show` — show help via video-cli

## Usage Examples
- "video-cli --help"
- "video-cli <args>"

## Installation

```bash
pip install video-cli
```
