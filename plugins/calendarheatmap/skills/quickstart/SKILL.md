# Calendarheatmap Plugin

Generate calendar heatmap visualizations from data.

## Commands

### Operations
- `calendarheatmap data visualize` — visualize data via calendarheatmap

## Usage Examples
- "calendarheatmap --help"
- "calendarheatmap <args>"

## Installation

```bash
go install github.com/nikolaydubina/calendarheatmap@latest
```

## Examples

```bash
calendarheatmap --version
calendarheatmap --help
```

## Key Features
- visualization\n- calendar\n- heatmap
