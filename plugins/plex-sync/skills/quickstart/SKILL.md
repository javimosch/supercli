# Plex-sync Plugin

Sync watched status between Plex servers.

## Commands

### Operations
- `plex-sync libraries sync` — sync libraries via plex-sync
- `plex-sync status show` — show status via plex-sync

## Usage Examples
- "plex-sync --help"
- "plex-sync <args>"

## Installation

```bash
npm install -g plex-sync
```

## Examples

```bash
plex-sync --version
plex-sync --help
```

## Key Features
- plex\n- media\n- sync
